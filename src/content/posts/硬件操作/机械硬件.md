# 电机选型
1. 有刷电机：
   - 原理：定子是磁铁(外)，转子是通电线圈组(内)
   - 缺点：有碳刷摩擦，噪音大，热量高易起火星，效率低，维护成本高
2. 无刷电机：
   - 原理：定子是通电线圈组(内)，转子是磁铁(外)，通过霍尔传感器检测转子位置再给定子特定线圈通电吸引转子磁铁
   - 优点：没有电刷摩擦，噪音小，运行平稳，效率高，寿命长，
   - 前提：需要电子调速器
   - 接线：
      1. 环控：闭环控制是一组三相线和一组排线，开环控制是一组三相线
      2. 电调：一组电源线和一组信号线，信号线一般是红线和黑线的CAN线，少数是红黄棕(S+-)或白黑(-S-)的PWM线
   - 参数：额定扭矩，额定转速，归零
   - 影响：电调安装位置和走线孔位
   - 常用：
      1. 大疆2006
         - 类型：减速电机（无刷电机+减速箱）
         - 优点：低扭矩，高效率，精确控制，低噪音
         - 参数：额定扭矩1N·m，1:36减速比
         - 电调：C610
         - 应用：小型拨弹弹盘，机械臂末端
      2. 大疆3508
         - 类型：减速电机（无刷电机+减速箱）
         - 优点：高扭矩，高效率，精确控制，低噪音
         - 参数：额定扭矩4N·m（带减速箱），1:19减速比（会把打印件搅碎）
         - 电调：C620
         - 应用：大型拨弹弹盘，摩擦轮
      3. 大疆6020
         - 类型：云台电机（无刷电机）
         - 优点：高精度，高速响应，高效能，轻量化，内置电调
         - 参数：额定扭矩1N·m
         - 电调：C610
         - 应用：舵轮底盘，云台转向
   - 特殊：
      1. 关节电机：
         - 优点：高精度，高扭矩，重复位置能力强，法兰输出
         - 品牌：小米CyberGear，达妙DM-J8009和4310-2EC
      2. 轮毂电机：
         - 优点：转子直接与车轮相连
         - 品牌：领空9025
      3. 无人机动力电机：
         - 优点：驱动无人机螺旋桨产生足够升力克服重力，效率高，高转速
         - 品牌：大疆E2000系统6010（PWM线），好盈X6plus（PWM线）
3. 伺服电机：
   - 小型：舵机
      1. 作用：只能在一定角度范围内转动
      2. 参数：距轴1cm位置能承载的重量kg/cm
      3. 缺点：精度差，扭矩小，有转动角度范围
   - 大型：工业伺服电机

## 1.1.螺栓
1. 连接类型
   - 普通螺栓连接：螺栓+螺母
   - 铰制孔用螺栓连接：铰制孔螺栓+螺母
   - 双头螺柱连接：双头螺柱+螺母
   - 螺钉连接：螺丝+螺纹孔
   - 紧定螺钉连接：紧定螺钉+螺纹孔
2. 螺栓
   - 分辨：配合螺母是螺栓，直接拧入是螺钉，日常统称是螺丝
   - 分类：螺头从厚到薄
      1. 杯头：不易滑丝（推荐1）
      2. 薄头：直径小且光滑
      3. 圆/盘头：圆形突起（推荐2）
      4. 倒边平头：稍厚稍扁
      5. 平头：接触平面大，容易滑丝（推荐4）
      6. 沉头：要配合沉头孔，接触面积大（推荐3）
   - 参数：
      1. 性能等级：推荐镀镍,等级高则抗拉强度高和硬度高和韧性差
      2. 直径：M2-2.5商品件,M3-5常用
      3. 长度：贯穿的通孔是零件高度+螺母高度+2mm，不贯穿的盲孔是<孔深
   - 常用：紧固从紧到松
      1. 螺栓+防松螺母：螺栓长度要略超出螺母
      2. 螺栓+弹簧垫圈+普通螺纹：
   - 防松：螺纹胶涂抹螺栓靠近尾部的一半螺纹（推荐中等强度可拆卸）
   - 设计：要求避免两段螺纹，螺头螺母不能与其他结构干涉，预留装配空间
3. 螺母
   - 分类：
      1. 普通螺母：无防松效果
      2. 防松螺母：工程塑料附着在螺纹上，螺栓和防松螺母配合时工程塑料挤压产生压力
      3. 拉铆螺母：压铆机压头无法压铆或抽芯方法无法达到效果且螺母装在外面
4. 垫片
   - 分类：
      1. 弹簧垫片：防松
      2. 金属平垫：增大受力面积，补偿空间距离，分隔两个相对转动的表面
      3. 金属轴套
      4. 尼龙垫片/柱：补偿空间距离
5. 铆钉
   - 工具：铆钉枪
   - 方式：通过铆钉枪抽芯，铆钉另一边会挤压变形而夹紧
   - 材料：铝，钢
   - 优点：紧固强，装配方便，重量轻
   - 缺点：无法拆卸
6. 垂直连接件
   - 分类：
      1. 六面螺母：精度1mm
      2. 四孔转接件
      3. 合页：精度3mm
      4. 角码
7. 金属柱子
   - 作用：垫高，加强板件间距，可以搭配螺栓或螺母使用
   - 分类：
      1. 铜柱：单通（公母）或双通（双母）
      2. 不锈钢柱：同理，但强度更高，价格更贵
   - 问题：精度不高常有误差
8. 铜嵌件
   - 作用：一般搭配3D打印件使用
   - 使用：加热3D打印件的孔并等待膨胀变形后挤进去嵌里面
9. 销
   - 分类：
      1. 圆柱销：定位，短轴传动
      2. 螺纹销：内/外螺纹销，固定后光轴端用于导向定位
      3. 开口销：和销轴配合用于轴向限位
      4. 销轴：
      5. 弹簧销：刚度大，安装后难拆卸，用于定位
## 1.轴系件
1. 轴承
   - 作用：垫在旋转轴和旋转支持部分，使旋转更顺畅，滚动摩擦减少损耗
   - 结构：金属防尘圈zz，胶盖（防水但阻尼大,无盖则润滑好），外圈，密封件，滚动体，保持架，内圈
   - 分类：
      1. 向心轴承：主要径向受力（压力），常见是深沟球轴承（如带挡边的法兰轴承，薄壁轴承，微型轴承）
      2. 推力轴承：只能轴向受力（推力），常见是推力球轴承和双向推力球轴承
      3. 向心推力轴承：径向和轴向受力（压力+推力），常见是角接触轴承和圆锥滚子轴承
      4. 关节/鱼眼轴承：外球面和内球面，适用于速度低的摆动运动
      5. 直线轴承：搭配线性导向轴使用，适用于直线运动导向机构
2. 塞打螺钉
   - 作用：转轴，定位销
   - 结构：螺头，光杆（高精度），螺纹
   - 使用：搭配防松螺母使用
3. 挡圈
   - 别名：卡簧
   - 作用：对轴进行轴向限位，轴向窜动量0.1-0.2mm，可以搭配波形垫圈补偿，可以承受一定力，但较大力会使挡圈崩离或崩断
   - 工具：轴用卡簧钳，孔用卡簧钳
   - 分类：
      1. 轴用挡圈：C型挡圈和E型挡圈（无需卡簧钳），搭配轴使用，需要在轴上切割出卡簧槽
      2. 孔用挡圈：搭配孔使用，需要在孔上切割出卡簧槽
4. 联轴器
   - 作用：连接电机和传动机构，补偿两轴线不对中的误差，分担电机承受的载荷和冲击
   - 分类：
      1. 胀紧联轴器：安装内环和轴不用拧紧，安装外环和轮毂后拧紧则楔形产生内外圈挤压力
      2. 夹紧联轴器：通过高强度拉力螺栓，抱紧内环和轴，而外环和轮毂则固定，同理安装完成后再拧紧
      3. 顶丝联轴器：用紧定螺钉固定同步带轮到电机轴上
5. 轴套
   - 作用：厚度很薄，保证只接触轴承内/外圈，减轻轴的磨损
   - 分类：
      1. 钢轴套
      2. 卷制轴套
      3. 石墨铜套：自润滑，可作为滑动轴承使用
## 1.工具
1. 扳手
   - 分类：
      1. 六角扳手：L型，拧内六角螺丝，球头边可以在一定角度下使用但容易滑丝所以不能大力（推荐维拉扳手但贵）
      2. 套筒扳手：套住六边形螺母
      3. 棘轮扳手：单向旋转，大力容易拧断螺栓，可以加套筒
      4. 开口扳手：拧螺母
      5. 电动扳手：省力且快速，但无法控制力度，在螺栓螺母压在打印件上和孔没完全对齐（产生装配应力）和安装精密昂贵零件时，不适用电动扳手
2. 钳子
   - 分类：
      1. 老虎钳：宽嘴，掰件
      2. 尖嘴钳：中宽嘴，掰件，剪切
      3. 针口钳：细嘴，深入狭小地方
      4. 斜口钳：大力剪切，例如玻纤板和铁丝等等
      5. 水口钳：剪切软物，例如导线等等
      6. 卡簧钳：安装卡簧，内卡簧钳是握柄合拢时钳口合拢，外卡簧钳是握柄合拢时钳口张开
3. 拉马
   - 作用：在轴承和轴配合很紧时（敲击进去的过盈配合），使用拉马，利用螺纹压力很大特性把轴承拉出
   - 使用：注意要分散压强
4. 拉铆枪
   - 作用：对铆钉抽芯
5. 角磨机
   - 作用：切割，打磨
6. 电钻
   - 作用：钻孔
7. 台虎钳
   - 作用：夹紧工件，利用夹紧力辅助装配和拆卸

# 1.材料工艺
1. FR-4玻纤板
   - 颜色：黑色
   - 特性：高刚度重量比，韧性强，耐燃，不导电，但能承受垂直板力小，抗弯能力一般
   - 厚度：常用2，3，4，5mm
   - 场合：小幅度抗弯，抗冲击，拉板，挠性零件（受弯曲能力），受力较小部位
2. 碳纤维板
   - 颜色：黑色
   - 特性：优异力学性能，强度高，重量轻，价格高，断面光滑，外部阻燃内部可燃，导电，加工粉尘有害
3. 3D打印件
   - 共性：不可绘制螺纹
   - 分类：
      1. PLA：3D打印机加热头熔化再凝固成型，力学性能一般，承压能力强，抗拉抗剪切能力弱（层与层之间分类），承受高冲击强度，受热易熔化，可以镶嵌铜螺母（使用电烙铁）
      2. PET：3D打印机加热头熔化再凝固成型，力学性能好，可以作为受力零件，但表面粗糙
      3. TPU：3D打印机加热头熔化再凝固成型，柔性材料，可以变形
      4. 光固化树脂：紫外光固化光敏树脂，力学性能较弱，软，壁厚不能过薄，长时间荧光日光充能会软化
4. 机床
   - 加工：雕铣机切割平面板（正视零件导出DXF格式）
      1. 内直角：圆柱刀头没有办法加工出内直角，可以在直角处割一个小圆槽（不推荐）
      2. 锐角：加工锐角时，回转时会产生力，可能会使板件翘起来，建议使用圆角过渡
      3. 孔位留余量：加工有误差，可以开大一点孔
      4. 圆直径：圆直径不能<圆柱刀头
5. 选择板件
   - 需求：结构件（强受力），连接件（弱受力），保护壳（受弯曲）
   - 镂空：必要结构设计，减重，防止干涉其他零件安装，方便走线，通过镂空便于装配或维修内部结构
   - 加固：使用其他结构件连接
