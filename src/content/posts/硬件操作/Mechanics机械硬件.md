# 2.标准件
## 2.1.紧固件
1. 螺栓螺钉：
   - 通名：配合螺母是螺栓，直接拧入是螺钉，日常统称是螺丝
   - 螺头形状分类：（螺头从厚到薄）
      1. 杯头：槽口宽深，不易滑丝（推荐1）
      2. 薄头：螺头直径小和光滑
      3. 圆/扁/盘/球头：圆形凸起，圆滑不会固连其他零件（推荐2,走线或转动件附近位置）
      4. 法兰面：增大受力面积，防松
      5. 凸肩：短光轴长螺纹，柱形沉头定位
      6. 倒边平头：微厚偏扁
      7. 平头：接触平面大。但容易滑丝（推荐4,高度受限位置）
      8. 沉头：接触面积大，锥形定位。但要搭配沉头孔（推荐3,空间受限位置）
      9.  紧定：没有螺头，尖尾，一般固定轴。但容易滑丝，需要轴有凹槽，负载弱
      10. 顶丝：没有螺头，平尾，一般固定轴
   - 螺头花型分类：
      1. 十字：
      2. 一字：
      3. 内六角：小型机械，受力好，不易滑丝
      4. 外六角：大型机械
      5. 梅花
      6. 防盗：
   - 螺纹分类：（加工方式）
      1. 手工丝锥螺纹
      2. 车制螺纹
      3. 滚压螺纹
      4. 自攻螺钉自攻丝（无需螺母和打孔攻丝）
   - 螺牙分类：（空缺形状）
      1. 三角形：60°等边三角形（公制螺纹:单位mm,螺距），普通连接作用，代号M
      2. 管螺纹：55°等腰三角形（英制螺纹:单位英寸,螺纹牙数），管件连接作用，代号G
      3. 梯形螺纹：30°梯形，传递双向力矩，传动作用，代号Tr
      4. 锯齿形螺纹：30+3°锯齿，传递单向力矩，传动作用，代号B
      5. 矩形螺纹：矩形，非标准螺纹，传动作用
   - 其他参数分类：
      1. 公称直径：螺纹大径（大径-牙顶,小径-牙底,中径-中间值）
      2. 线数：单线，双线，多线
      3. 螺距：粗牙，细牙
      4. 导程：导程（旋转一圈位移）=线数×螺距（两相邻螺牙间距）
      5. 旋向：顺进逆出是右螺旋纹，无人机一般是左旋螺纹
      6. 牙型：全牙（全小径）、半牙（大径+小径）（位移小）
   - 选购参数：
      1. 代号：螺牙分类×公称直径×螺距×螺杆长度
      2. 公称直径：商品件常用2-2.5,项目常用3-5
      3. 螺杆长度：嵌入长度。贯穿通孔是零件高度+螺母高度+2mm，不贯穿盲孔是<孔深
      4. 性能等级：推荐镀镍，<8.8的普通不锈钢螺栓，>8.8的低碳合金钢的高强度螺栓，推荐使用12.9级螺栓（小数点前数字×100=最小抗拉强度N/mm²,小数点后数字×小数点前数字×10=屈服强度N/mm²）。等级高则抗拉强、硬度高。但韧性差
      5. 制作精度：A（精制>8.8性能等级）、B（精制>8.8性能等级）、C（粗制-临时固定<8.8性能等级）
   - 设计：
      1. 避免两端螺纹
      2. 螺头螺母不与其他结构干涉
      3. 预留装配空间
      4. 开孔比螺丝大1-2mm（不能定位）
   - 装配：
      1. 螺栓+防松螺母(+普通螺纹孔)：螺栓长度要略超出螺母（强紧固,均匀预紧力）
      2. 螺栓+螺纹胶+普通螺纹孔：螺纹胶推荐中等强度蓝胶可拆卸（绿胶≈红胶(常用)(干后粉末)>蓝胶(常用)(干后糊状)≈防松螺母>紫胶>弹垫）。靠近螺栓尾部的半螺纹涂抹螺纹胶（强紧固）
      3. 螺栓+弹簧垫圈+普通螺纹孔：（一般紧固,不均匀预紧力）
2. 螺柱：
   - 作用：垫高，加强板件间距
   - 分类：（分为单通(公母)和双通(双母)）
      1. 尼龙：分类有圆柱形，六角柱。强度一般，不导电，不导磁
      2. 铝柱：分类有圆柱形，六边形头，六角柱。强度低会形变，用于垫高
      3. 铜柱：强度一般，价格便宜，但精度差
      4. 不锈钢柱：强度高，价格便宜。但精度差
3. 螺母：
   - 分类：
      1. 六角螺母：手能拧动，无防松效果
      2. 防松螺母：尼龙圈附着在螺纹上，在螺栓和防松螺母配合时尼龙圈会挤压产生压力
      3. 拉铆螺母：（螺母在外，在压铆机无法压铆和铆钉枪无法抽芯时使用）使用拉铆枪把螺母嵌入工件的孔，不方便拆装
      4. K型螺母：爪子可以嵌入工件表面，一般在打印件使用，轻微损害零件
      5. 法兰螺母：凹凸法兰面帮助抓紧工件表面
      6. 船形螺母：（T型滑块螺母）一般卡在铝型材槽口
      7. 铜嵌螺母：（热熔螺母）放置铜嵌螺母小孔在3D打印件的孔里，使用电烙铁搭在铜嵌螺母大孔上，实现间接加热3D打印件的孔，等待膨胀变形后嵌入，嵌入一半后可以翻转包含铜嵌螺母的零件扣在桌子上保证平齐/使用螺丝头压平齐。受力差容易连同螺栓掉落，更推荐打印件凹刻放置防松螺母
      8. 六面螺母：六面都可以作为螺母，精度1mm，螺纹容易滑丝
      9. 蝶形手拧螺母：手拧螺母，便于后续手调
      10. 夹板子母螺栓螺母：螺母和螺栓夹住工件
4. 垫圈垫片：
   - 分类：
      1. 弹簧垫片：回弹力防松（主要用于没有防松螺母位置），吸收冲击和振动
      2. 金属平垫：增大受力面积，补偿空间距离，分隔两个相对转动表面减少摩擦
      3. 尼龙垫片/柱：补偿空间距离，容易变形，不导电
      4. 止退垫片：
      5. 双叠防松垫片：
      6. 鞍形垫圈：回弹力防松
      7. 橡胶垫圈：密封，减震
      8. 绝缘垫圈：绝缘
5. 铆钉：
   - 材料：铝（强度低），钢（难拆卸）
   - 特点：紧固强，重量轻，装配方便。但无法拆卸
   - 装配：通过铆钉枪抽芯，铆钉另一边会挤压变形而夹紧
   - 功能分类：
      1. 实心：受力大的金属连接（常用）
      2. 空心：受力小的薄板或非金属连接
   - 安装分类：
      1. 热铆：紧密性好。但铆杆和钉孔有间隙，不能传递力矩
      2. 冷铆：铆杆和钉孔无间隙
   - 装配分类：
      1. 拉铆（拉铆枪）：铝制，可以用钻头暴力拆卸
      2. 敲铆（锤子敲）：不锈钢制，不可拆卸
   - 应用分类：
      1. 活动铆接：可以互相转动，不是刚性连接
      2. 固定铆接：不能相互活动，是刚性连接
      3. 密封铆接：不漏气液
   - 尺寸：
      1. 开口型边缘头抽芯铆钉：常用
      2. 沉头抽芯铆钉：需要表面平滑
      3. 封闭型抽芯铆钉：适用高负载和密封
6. 分体夹紧式法兰：
   - 结构：两部分金属弯片构成

## 2.2.传动件
1. 齿轮齿条
   - 定义：轮缘有齿，能连续啮合传递运动和动力
   - 参数：模数(决定齿大小)，齿数（>17避免根切,齿数多则运动平稳），齿宽（小齿轮比大齿轮宽），传动比
   - 设计：分度圆相切，渐开线，啮合线与切线，压力角（一般20°）
   - 公式：模数=分度圆齿距÷π，分度圆直径=模数×齿数，齿顶圆直径=模数×(齿数+2)，分度圆齿距=分度圆齿厚+分度圆齿槽宽
   - 特点：精度高。但价格昂贵
   - 装配：
      1. 直齿轮-直齿条：精度高，速度快，可靠强，承载力大，运动平稳，直线传动
      2. 直齿轮-直齿轮：效率高，传动比大（1-8），加工方便，价格便宜。但噪音大，重合度低
      3. 斜齿轮-斜齿轮：噪音小，运动平稳，重合度高，啮合长度长可高速传动。但螺旋角会产生轴向力
      4. 圆锥直齿轮-圆锥直齿轮：交错轴传动，传动比大（1-8）。但要求安装精度高，轴承要承担轴向径向载荷，轴系设计复杂
      5. 圆锥螺旋齿轮-圆锥螺旋齿轮：交错轴传动，重合度高。但要求安装精度高，要求安装位置不易变形
2. 蜗杆蜗轮
3. 丝杆-螺母
   - 特点：精度高，运动平稳，转动变直线运动，负载大
   - 分类：
      1. 丝杆-梯形螺母：
         - 原理：相互螺旋运动实现直线运动
         - 特点：精度高，负载大，有自锁功能固定位置（条件是螺纹升角<静摩擦角）。但效率低（摩擦阻力大）
      2. 丝杆-滚珠螺母：
         - 原理：滚珠和丝杆螺纹配合实现直线运动
         - 特点：效率高，精度高，负载大，噪音低，运动平稳，可高速传动。但价格昂贵，不能自锁，需要维护保养
4. 带-轮
   - 分类：
      1. 同步带-同步轮
         - 原理：主动轮通过同步带带动从动轮旋转
         - 特点：适合距离短，重量轻，精度高，传动比准确，运动平稳，保养方便，工作安静，可传递转矩，可使用压板压紧同步带实现直线运动。但弹性大易变形（在安装时不能太紧），可能会跳齿，对中心距和尺寸稳定性要求高，需要使用张紧轮张紧同步带
      2. 链条-链轮
         - 特点：适合距离长，负载大，可靠强，刚度大不易变形（在安装时要紧），适合各类恶劣工况。但重量重，效率低，噪音大，价格昂贵，容易磨损，容易伸长，运动不平稳（不适用在急速反向传动），需要使用张紧轮或楔形块张紧链条
   - 张紧装置：（尽量对称设计保证张力一致）
      1. 定期张紧装置：定期拧紧调节螺栓增大两轮中心距
      2. 张紧轮张紧装置：增加带轮压紧或撑紧带
      3. 自动张紧装置：添加电机依赖重力作用自动下压张紧带
5. 联轴器：
   - 作用：连接电机和传动机构，补偿两轴线不对中的误差，分担电机承受的载荷和冲击
   - 工作分类：
      1. 刚性联轴器：
         - 结构：两个楔形圆环和螺钉
         - 特点：负载大，精度高，拆卸方便，结构简单，价格便宜，两轴转速严格相等。但两轴严格对中，没有补偿相对位移能力，没有缓冲减震能力
         - 常用
            1. 胀紧套（胀紧联结套）
               - 原理：拧紧螺丝则孔内缩而轴外涨（注意额定扭矩），在内环与轴之间和在外环与轮毂之间都产生巨大抱紧力可以传递扭矩，实现轴和轮毂的无键连接
               - 特点：负载大，可靠强，运动平稳，装拆方便，对中精度高
               - 装配：对角交叉拧紧（先拧缺口一端,再拧对角,再拧缺口另一端,再拧对角,最后拧剩下），手拧一遍再≈3-5N·m定扭矩扳手拧一遍，拆卸同理对角交叉拧松
            2. D型孔夹紧式联轴器
               - 原理：夹紧轴再固定工件
               - 装配：注意光孔和螺纹孔位置
         - 尺寸：常用胀紧套，法兰联轴器（一端法兰一端孔,推荐不要用顶丝固定）
      2. 挠性联轴器：
         - 结构：两圆环和工程塑料（两圆环分别和工程塑料连接）/铝合金/弹簧/机械结构
         - 特点：两轴转速基本一致，可以补偿相对位移能力，一定缓冲减震能力。但负载小（负载太大会损坏弹性元件）
         - 尺寸：常用膜片联轴器（连接处有膜片,推荐用夹紧式），梅花弹性联轴器（连接处有塑料块,推荐用夹紧式），铝合金弹性联轴器（连接处是弹簧状,推荐用夹紧式），十字轴式万向联轴器（连接处是两个正交铰链,推荐用夹紧式）（补偿能力强但结构复杂和两轴转速可能不一致）（刚性），弹簧联轴器（易坏不推荐）
   - 装配分类：
      1. 胀紧联轴器：安装内环和轴不用拧紧，安装外环和轮毂后拧紧则楔形产生内外圈挤压力
      2. 夹紧联轴器：（缩小孔直径夹紧）通过高强度拉力螺栓夹紧轴实现连接，而外环和轮毂则固定，同理安装完成后再拧紧
      3. 顶丝联轴器：（扭紧螺丝顶紧轴,轴易滑和螺丝滑牙而不推荐）通过紧定螺钉压紧轴实现连接
6. 键
   - 作用：传递运动和扭矩，同步旋转，保证同轴度
   - 分类：
      1. 普通平键
      2. 半圆键
      3. 钩头楔键
      4. 外花键
      5. 内花键
7. 固定环
   - 分类：
      1. 光轴止动环：小轴，轴和环同步旋转止动
      2. 开口型固定环：大轴

## 2.3.承导件
1. 滚动轴承：
   - 作用：垫在旋转轴和旋转支持部分，把轴间原本的滑动摩擦力转化为轴承内球的滚动摩擦，滚动摩擦减少损耗而旋转顺畅
   - 结构：双侧橡胶密封圈2RS（防水但阻尼大）/金属防尘盖ZZ（无则润滑好），外圈，密封件，滚动体（球轴承,滚子）（单列,双列），保持架，内圈
   - 方向
      1. 轴向：沿轴的圆柱体旋转中心轴的方向
      2. 径向：沿轴的截面经过轴心的方向
   - 分类：
      1. 向心轴承：
         - 受力：主要径向受力（压力），10%轴向受力，不能倾覆受力
         - 结构：两个圆环、中间滚动体和中间保持架，不可分开
         - 尺寸：常见是深沟球轴承（例如带挡边的法兰轴承、薄壁轴承、微型轴承）（双列深沟球轴承,外球面球轴承）（点接触）、圆柱滚子轴承（线接触）、调心滚子轴承、调心球轴承
      2. 推力轴承：
         - 受力：只能轴向受力（推力），而且要持续压住
         - 结构：两个圆环片（在静止件的松环外环是内径大,在运动件的紧环内环内径小0.1-0.3mm）、中间滚动体和中间保持架，外圈较大固定工件内圈较小配合轴，可分开
         - 尺寸：常见是推力球轴承（点接触）、推力圆柱滚子轴承（硬质光滑工件可以省略但不推荐）（线接触）、双向推力球轴承、推力滚针轴承（针状薄）、推力调心滚子轴承
      3. 向心推力轴承：
         - 受力：径向和轴向受力（压力+推力）
         - 结构：两个圆环、倾斜中间滚动体和中间保持架，部分可分离
         - 尺寸：常见是角接触轴承（双列角接触球轴承,四点接触球轴承）（点接触）、圆锥滚子轴承（线接触）、交叉滚子轴承（精度高,刚度高,全向负载,轴间间隙可调整）
      4. 直线轴承：搭配线性导向钢轴使用，适用于直线运动导向机构。精度高，摩擦小，运动平稳。但负载低，高速运动噪音大振动大，对轴光滑度和硬度要求高
      5. 关节/鱼眼轴承：外球面和内球面，适用于速度低的空间旋转摆动运动
   - 设计：接触工件不能同时碰到内外圈，区分旋转部分和静止部分，尽量使用凸缘挡边法兰轴承（方便定位固定）或薄壁轴承（内外圈距离薄）或交叉滚子轴承（双向力），单向轴承推荐成对使用，≈10mm小轴用球>20mm大球用滚子，径向极限位置用薄壁或滚针，轴向极限位置用推力滚子中间层，推荐组合多个轴承使用
   - 装配：台虎钳压入，拉马拔出 
   - 代号：轴承类型-尺寸型号-内径型号（根据需求尺寸查表确定代号）
   - 检查：相对滑动阻力很小，无晃动（过盈配合）（有晃动用680圆柱体胶固定），生锈用除锈剂，不够滑用润滑油
   - 装配：轴承胶
2. 滑动轴套：
   - 作用：厚度很薄保证只接触轴承内/外圈，内外圈光滑而减轻轴的磨损，减震减噪

## 3.1.结构件
1. 管材（1维-长度）
   - 原理：管柱发现内部掏空仍有不错的结构强度，而且还节省质量和提供更多空间
   - 特点：重量轻，强度高，刚度高变形小
   - 加工：焊接，打孔（一般加工孔比尺寸大），镂空（需要仿真强度分析）
   - 尺寸：长×宽×壁厚
   - 分类：
      1. 铝方管
         - 尺寸：常用面积30×30mm厚度1mm，增加强度优先考虑增加长宽
         - 特点：力学性能好，结构简单，价格便宜，重量轻。但抗压和耐热易变形，不可加工螺纹孔
         - 装配：
            1. 直接在铝方管上打贯穿孔后用螺栓螺母连接
            2. 使用夹具夹紧铝方管后焊接，但要注意铝管受热容易变形
            3. 使用拉铆螺母铆接在铝方管上后有螺纹就可以用螺栓连接（注意拉铆螺母厚度）
            4. 贯穿铝方管螺栓需要添加垫片防止用力过渡而表面凹陷
         - 加强：塞入打印件，垫入玻纤板
      2. 碳管
         - 尺寸：常用碳圆管，不常用碳方管（不易连接）
         - 特点：强度高，刚度高，重量轻，精度高。但价格昂贵，不易打孔
         - 装配：通过夹紧方式连接
      3. 型铝
         - 加工：通过挤压得到的铝型材
         - 特点：力学性能好，价格便宜。但重量重，装配精度差
         - 装配：搭配T型螺母使用
2. 板材（2维-长宽）
   - 设计：倒圆角防止应力集中
   - 分类：
      1. 碳纤维板
         - 颜色：黑
         - 特点：力学性能好，强度高，重量轻，精度高，断面光滑。但价格昂贵，导电，外部阻燃内部可燃，加工粉尘有害，不能承受侧面打孔
         - 重量：≈1800kg/m³
         - 加工：打孔不要打柱形沉头孔而是锥形沉头孔（层间受力较弱），一般加工孔比尺寸小0.1mm,价格约为玻璃纤维板的2倍，发货时间5-10天
         - 尺寸：常用厚度1（不受力且无刚度要求）、2（受力10kg量级）、3、4（受力100kg量级的结构件）、5、6、7、8（受力较大且刚度要求高）mm
      2. FR-4玻璃纤维板
         - 颜色：黑，黄，绿
         - 特点：强度高，韧性好，耐燃好，重量轻（比碳板略重），不导电，价格便宜。但抗弯差
         - 重量：≈2500kg/m³
         - 加工：雕铣机高速旋转铣刀平面加工，原料便宜加工贵，发货时间5-10天
         - 尺寸：常用厚度2、3、4（结构件）、5mm
         - 设计：位置在受力小、抗冲击、接受微弯变形、拉板
      3. 亚克力
         - 特点：透光，价格便宜，可以加工平面或平面弧。但强度低，材料软
         - 重量：≈1200kg/m³
         - 加工：一般加工孔比尺寸大0.1mm
         - 尺寸：常用厚度4、5mm
3. 机加工件（3维-立体）
   - 材料：常用6061铝合金，304不锈钢
   - 费用：材料费+加工费（费用与设计图合理规划相关）
   - 加工：通过车（工件旋转,刀具运动,加工回转表面）铣（工件运动,刀具旋转,加工平面曲面）钻镗刨磨和钣金和线切割等方式对材料进行去除多余的加工而得到工件
   - 特点：力学性能好，精度高，表面光滑，可加工螺纹孔。但加工图纸要标公差（孔+轴-），价格昂贵
   - 设计：
      1. 孔离边远：避免刀头无法伸入加工
      2. 孔无斜面：孔的两侧都应该是平行面，避免钻入钻出受力不一致导致钻头偏斜
      3. 配合凸台：配合面要精加工处理但面积越大越贵，添加小凸台则只需要加工小凸台
      4. 孔深不长：孔深太长容易断钻头
      5. 薄壁不深：提高加工难度降低良品率
      6. 避免内槽：难加工
   - 加工商：铨洲
4. 3D打印件（3维-立体）
   - 特点：价格便宜。但强度低，表面粗糙，
   - 参数：
      1. 密度：过小则脆弱，过大则浪费
   - 分类：
      1. 加热头熔化材料再凝固的逐层打印
         - PLA（聚乳酸）：抗冲击好，抗层压好，力学性能一般，可嵌螺母（使用电烙铁）。但抗拉差，抗侧压差，受热易熔
         - PET：力学性能好。但表面粗糙
         - TPU：柔软可变形，抗冲击强
         - Nylon（尼龙）：力学性能较好。但表面粗糙
      2. 紫外光固化光敏树脂的整个打印
         - 光固化：精度高，温度不敏感，表面光滑。但力学性能弱，柔软，长时间日光或荧光会软化变形，只能让加工商嵌入螺母
   - 注意：
      1. 壁厚不薄：壁厚>2mm，否则影响力学性能，打印成败
      2. 孔留余量：由于打印机精度和冷却缩小的问题，孔要预设大一些，至少余量>0.4mm
      3. 打印方向：层侧力学性能<层面，层侧纹理方向影响运动阻力，方便拆卸支撑
      4. 加强强度：可以塞入圆柱销等工件加强打印件强度
      5. 不印螺纹：因为材料硬度不够，所以不能打印螺纹孔用于固定，但可以打印放置螺母的孔位
5. 赛钢聚甲醛POM
   - 特点：强度高，刚度高，自润滑
6. 聚氨酯
   - 特点：摩擦阻力大

      1. 摩擦轮304不锈钢机加工件加上聚氨酯包胶

# 6.设计
## 6.1.轴系设计
1. 欧拉角
   - Pitch：俯仰角，绕X轴旋转，点头
   - Yaw：偏航角，绕Y轴旋转，摇头
   - Roll：翻滚角，绕Z轴旋转，沿前进方向旋转
## 6.2.结构设计
设计：固定后只保留需要自由度而其他不能动（轴向、径向、周向），使用轴承分隔开相互摩擦位置，保证结构强度，分析受力选择零件，清楚哪部分固定哪部分活动，安装位置余量，解耦模块化，考虑未来迭代的方便设计，强度（抵抗破坏）刚度（形变程度）精度（公差标注），避免应力集中（零件表面出现圆孔槽口,可以使用不锈钢垫片或玻纤板增加局部厚度），考虑螺丝疲劳（受压紧力和压力,薄板螺纹孔径3-5mm才够用），避免悬臂梁结构而是双端固定，考虑螺栓安装在孔里有活动余量所以要用定位销（定位销定位而螺栓是紧固）或者使用塞打螺栓，余量>3-5mm
- 需求：结构件（强受力），连接件（弱受力），保护壳（受弯曲）
- 镂空：必要结构设计，在不受力位置设置，减重轻量化，避免干涉其他零件安装，方便走线，通过镂空便于装配或维修或观察内部结构，要注意保留基础结构和主要受力支撑，推荐倒圆角的内或内梯形镂空
- 加固：使用其他结构件连接
## 6.3.紧固设计
1. 普通螺栓连接：螺栓+螺母
2. 铰制孔用螺栓连接：铰制孔螺栓+螺母
3. 双头螺柱连接：双头螺柱+螺母
4. 螺钉连接：螺丝+螺纹孔
5. 紧定螺钉连接：紧定螺钉+螺纹孔
## 6.4.加工设计
- 格式：DXF（正视零件导出）
- 加工：雕铣机切割平面板
   1. 内直角：圆柱刀头没有办法加工出内直角，可以在直角处割一个小圆槽（不推荐）
   2. 锐角：加工锐角时，回转时会产生力，可能会使板件翘起来，建议使用圆角过渡
   3. 孔位留余量：加工有误差，可以开大一点孔s
   4. 圆直径：圆直径不能<圆柱刀头
## 6.5.软件设计
1. 有限元分析
2. 迈迪（Solidworks插件）
   - 圆柱齿轮设计

## 6.6.公差设计
1. 基本偏差（公差带区间位置）：孔大写（要大），轴小写（要小）
2. 标准公差等级代号（公差带大小）：
3. 加工：精度越高，加工要求越高，价格越昂贵
4. 配合：间隙配合，过渡配合（公差带有重合），过盈配合
## 6.7.受力分析
1. 轴内拉伸/压缩
2. 剪切
3. 扭转
4. 平面弯曲
## 6.8.工件能力
1. 强度：抵抗破坏能力
2. 刚度：受力形变范围限制
3. 稳定：抵抗外界干扰能力
## 6.9.有限元分析
1. 应力：一点处的内力聚集程度，越集中该点越容易破损
2. 分析图：（小）蓝-绿-黄-红（大）
3. 应力集中：几何形状急剧变化位置，例如孔、槽、缺口、焊点
4. 解决办法：避免尖角，流线型外形，增加倒/圆角或渐变过渡
## 6.10.工件功能
1. 结构件：受力大，强度大，刚度大，4-6mm板件
2. 连接件：连接工件，刚度一般，2-3mm板件
3. 保护壳：1mm不镂空碳板，2mm镂空玻纤板
## 6.11.工件设计
1. 板件间拼插精度低（内角位置需要释放孔）
2. 孔位留余量（商加工1mm,自加工2mm）
3. 避免内角角度过小（在加工锐角时急停急转容易断刀）
4. 避免<2mm位置（刀具直径2mm）
## 6.12.零件命名
1. 淘宝：TB-商品名称
2. 标准件：STD-类型-尺寸
3. 板材：NC-厚度-GF/CF/AL-板材
4. 打印件：PLA/PET/SLA-打印件
5. 铝方管：AP-截面长×宽×厚-铝方管
6. 碳圆管：CP-截面外径×厚度-碳圆管
7. 机加工件：CNC/POM-机加工件

## 6.13.传动设计
1. 误差：电机直驱是最优解，电机内部有虚位+传动机构装配误差=电控调试困难，而且减少机械设计困难和成本
2. 场合：
   - 回转运动转变成直线运动
   - 没有放置电机的位置
   - 扭矩或负载（传动比）不符
## 6.14.电机设计
1. 区分活动部件和固定部件
2. 活动部件连接电机转子，固定部件连接电机定子
3. 活动部件不与固定部件干涉
4. 电机走线
## 6.15.运动设计
1. 自由度：XYZ的平移和旋转
2. 抽象具体：先用点、线、圆来代替具体构建，再用旋转平移相互联系，然后根据运动规律设计参数，接着根据具体要求设计细节，最后选择合适的驱动动力件

2. 板材堆叠+支撑件
3. 板材+管材+角接件
4. 榫卯连接
