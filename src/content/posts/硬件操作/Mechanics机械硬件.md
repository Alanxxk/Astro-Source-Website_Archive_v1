# 2.标准件
## 2.1.紧固件
1. 螺栓：
   - 通名：配合螺母是螺栓，直接拧入是螺钉，日常统称是螺丝
   - 分类：（螺头从厚到薄）
      1. 杯头：不易滑丝（推荐1）
      2. 薄头：螺头直径小和光滑
      3. 圆/扁头：圆形凸起（推荐2）
      4. 倒边平头：微厚偏扁
      5. 平头：接触平面大。但容易滑丝（推荐4）
      6. 沉头：接触面积大。但要搭配沉头孔（推荐3）
   - 参数：
      1. 直径：商品件常用M2-2.5，项目常用M3-5
      2. 长度：贯穿通孔是零件高度+螺母高度+2mm，不贯穿盲孔是<孔深
      3. 性能等级：推荐镀镍。等级高则抗拉强、硬度高。但韧性差
   - 设计：
      1. 避免两端螺纹
      2. 螺头螺母不与其他结构干涉
      3. 预留装配空间
      4. 开孔比螺丝大1-2mm
   - 装配：
      1. 螺栓+防松螺母(+普通螺纹)：螺栓长度要略超出螺母（强紧固）
      2. 螺栓+螺纹胶+普通螺纹：推荐中等强度可拆卸。靠近螺栓尾部的半螺纹涂抹螺纹胶（强紧固）
      3. 螺栓+弹簧垫圈+普通螺纹：（一般紧固）
2. 螺母：
   - 分类：
      1. 普通螺母：无防松效果
      2. 防松螺母：工程塑料附着在螺纹上，在螺栓和防松螺母配合时工程塑料会挤压产生压力
      3. 拉铆螺母：螺母在外，在压铆机无法压铆和铆钉枪无法抽芯时使用
3. 垫片：
   - 分类：
      1. 弹簧垫片：防松
      2. 尼龙垫片/柱：补偿空间距离
      3. 金属平垫/轴套：增大受力面积，补偿空间距离，分隔两个相对转动表面
4. 铆钉：
   - 材料：铝，钢
   - 特点：紧固强，重量轻，装配方便。但无法拆卸
   - 装配：通过铆钉枪抽芯，铆钉另一边会挤压变形而夹紧
5. 销：
   - 分类：
      1. 圆柱销：定位，短轴传动
      2. 螺纹销：有内、外螺纹销，固定后光轴端用于导向定位
      3. 弹簧销：定位。不易变形，但难拆卸
      4. 开口销-销轴：轴向限位
6. 铜嵌件：
   - 装配：加热3D打印件的孔等待膨胀变形后嵌入
7. 金属螺柱：
   - 作用：垫高，加强板件间距
   - 分类：（分为单通(公母)和双通(双母)）
      1. 铜柱：强度一般，价格便宜，但精度差
      2. 不锈钢柱：强度高，但价格昂贵，精度差
8. 垂直连接件：
   - 分类：
      1. 六面螺母：精度1mm
      2. 四孔转接件：
      3. 合页：精度3mm
      4. 角码：

## 2.3.传动件
1. 齿轮齿条
   - 定义：轮缘有齿，能连续啮合传递运动和动力
   - 参数：模数(齿大小)，齿数，齿厚
   - 设计：分度圆，渐开线，啮合线与切线，压力角
   - 公式：分度圆直径=模数×齿数，齿顶圆直径=模数×(齿数+2)
   - 装配：
      1. 直齿轮-直齿条：精度高，速度快，可靠强，承载力大，运动平稳，直线传动
      2. 直齿轮-直齿轮：效率高，传动比大（1-8），加工方便，价格便宜。但噪音大，重合度低
      3. 螺旋齿轮-螺旋齿轮：噪音小，运动平稳，重合度高，可高速传动。但会产生轴向力
      4. 圆锥直齿轮-圆锥直齿轮：交错轴传动。但要求安装精度高，轴承要承担轴向径向载荷，轴系设计复杂
      5. 圆锥螺旋齿轮-圆锥螺旋齿轮：交错轴传动，重合度高。但要求安装精度高，要求安装位置不易变形
2. 蜗杆
3. 丝杆
   - 分类：
      1. 丝杆-梯形螺母：
         - 原理：相互螺旋运动实现直线运动
         - 特点：精度高，负载大，有自锁功能固定位置（条件是螺纹升角<摩擦角）。但效率低（摩擦阻力大）
      2. 丝杆-滚珠螺母：
         - 原理：滚珠和丝杆螺纹配合实现直线运动
         - 特点：效率高，精度高，负载大，噪音低，运动平稳，可高速传动。但价格昂贵，不能自锁，需要维护保养
4. 带
   - 分类：
      1. 同步带-同步轮：
         - 原理：主动轮通过同步带带动从动轮旋转
         - 特点：传动比准确，可传递转矩，可使用压板实现直线运动。但可能会跳齿，对中心距和尺寸稳定性要求高，需要张紧同步带
5. 链
   - 分类：
      1. 链条-链轮：
         - 特点：负载大，距离长，可靠强。但效率低，需要张紧链条
## 2.4.轴系件
1. 轴承：
   - 作用：垫在旋转轴和旋转支持部分，滚动摩擦减少损耗而旋顺畅
   - 结构：双侧橡胶密封圈2RS（防水但阻尼大）/金属防尘盖ZZ（无则润滑好），外圈，密封件，滚动体，保持架，内圈
   - 分类：
      1. 向心轴承：主要径向受力（压力），常见是深沟球轴承（例如带挡边的法兰轴承、薄壁轴承、微型轴承）
      2. 推力轴承：只能轴向受力（推力），常见是推力球轴承和双向推力球轴承
      3. 向心推力轴承：径向和轴向受力（压力+推力），常见是角接触轴承、圆锥滚子轴承和交叉滚子轴承
      4. 关节/鱼眼轴承：外球面和内球面，适用于速度低的摆动运动
      5. 直线轴承：搭配线性导向轴使用，适用于直线运动导向机构
2. 塞打螺钉：
   - 作用：转轴，定位销
   - 结构：螺头，光杆（精度高），螺纹
   - 装配：搭配防松螺母使用
3. 挡圈：
   - 通名：卡簧
   - 作用：对轴进行轴向限位（精度0.1-0.2mm），可以搭配波形垫圈补偿，可以承受一定力但大力会崩离崩断挡圈
   - 分类：
      1. 轴用挡圈：C型挡圈和E型挡圈（无需卡簧钳），搭配轴使用，需要在轴上切割出卡簧槽
      2. 孔用挡圈：搭配孔使用，需要在孔上切割出卡簧槽
   - 装配：使用轴用卡簧钳、孔用卡簧钳安装
4. 联轴器：
   - 作用：连接电机和传动机构，补偿两轴线不对中的误差，分担电机承受的载荷和冲击
   - 分类：
      1. 胀紧联轴器：安装内环和轴不用拧紧，安装外环和轮毂后拧紧则楔形产生内外圈挤压力
      2. 夹紧联轴器：通过高强度拉力螺栓，抱紧内环和轴，而外环和轮毂则固定，同理安装完成后再拧紧
      3. 顶丝联轴器：使用紧定螺钉固定同步带、同步轮到电机轴上
   - 尺寸：
      1. 
5. 轴套：
   - 作用：厚度很薄保证只接触轴承内/外圈，减轻轴的磨损
   - 分类：
      1. 钢轴套：
      2. 卷制轴套：
      3. 石墨轴套：自润滑，可以作为滑动轴承

# 3.加工件
1. 3D打印件
   - 格式：STL
   - 参数：
      1. 密度：过小则脆弱，过大则浪费
   - 分类：
      1. 加热头熔化材料再凝固
         - PLA：抗冲击好，抗层压好，力学性能中，可嵌螺母（使用电烙铁）。但抗拉差，抗侧压差，受热易熔
         - PET：力学性能好。但表面粗糙
         - TPU：柔软可变形
      2. 紫外光固化光敏树脂
         - 光固化：精度高，温度不敏感，表面光滑。但力学性能弱，柔软，长时间日光或荧光会软化变形，只能让加工商嵌入螺母
   - 注意：
      1. 壁厚不薄：否则影响力学性能，打印成败
      2. 孔留余量：由于打印机精度和冷却缩小的问题，孔要预设大一些
      3. 打印方向：层侧力学性能<层面
      4. 加强强度：可以塞入销等工件加强打印件强度
      5. 不印螺纹：因为材料硬度不够，所以不能打印螺纹孔用于固定
2. 机加工件
   - 材料：常用6061铝合金
   - 加工：通过车铣刨磨等方式对材料进行去除多余的加工而得到工件
   - 特点：力学性能好，精度高，表面光滑，可加工螺纹孔。但加工图纸要标公差，价格昂贵
   - 设计：
      1. 孔离边远：避免刀头无法伸入加工
      2. 孔无斜面：孔的两侧都应该是平行面，避免钻入钻出受力不一致导致钻头偏斜
      3. 配合凸台：配合面要精加工处理但面积越大越贵，添加小凸台则只需要加工小凸台
      4. 孔深不长：孔深太长容易断钻头
3. FR-4玻纤板
   - 颜色：黑，黄，绿
   - 特点：韧性好，耐燃好，重量轻，不导电。但抗弯差
   - 尺寸：常用厚度2、3、4（结构件）、5mm
   - 设计：位置在受力小、抗冲击、接受微弯变形、拉板
4. 碳纤维板
   - 颜色：黑
   - 特点：力学性能好，强度高，重量轻，断面光滑。但价格昂贵，导电，外部阻燃内部可燃，加工粉尘有害
5. 碳管
   - 尺寸：常用碳圆管，不常用碳方管（不易连接）
   - 特点：强度高，重量轻。但价格昂贵
   - 装配：通过夹紧方式连接
6. 铝方管
   - 尺寸：常用面积30×30mm厚度1mm
   - 特点：力学性能好，结构简单，价格便宜，重量轻。但抗压和耐热易变形，不可加工螺纹孔
   - 装配：
      1. 直接在铝方管上打贯穿孔后用螺栓螺母连接
      2. 使用夹具夹紧铝方管后焊接
      3. 使用拉铆螺母铆接在铝方管上后有螺纹就可以用螺栓连接（注意拉铆螺母厚度）
   - 加强：塞入打印件，垫入玻纤板
7. 型铝
   - 加工：通过挤压得到的铝型材
   - 特点：力学性能好，价格便宜。但重量重
   - 装配：搭配T型螺母使用
# 4.工具
1. 扳手
   - 分类：
      1. 六角扳手：拧内六角螺丝，L型，球头边可以在一定角度下使用但容易滑丝（推荐维拉扳手但价格昂贵）
      2. 套筒扳手：套住外六角螺丝
      3. 棘轮扳手：单向旋转但大力容易拧断螺栓，可以添加套筒
      4. 开口扳手：拧螺母或固定螺母
      5. 电动扳手：快速且省力但无法控制力度，安装精密零件或打印件或孔不对齐（产生装配应力）则不适用
2. 钳子
   - 分类：
      1. 老虎钳：宽嘴，掰件
      2. 尖嘴钳：中宽嘴，掰件，剪切

      4. 斜口钳：大力剪切，适合材料是玻纤板、铁丝等等
      5. 水口钳：剪切软物，适合材料是导线等等
      6. 卡簧钳：安装卡簧，内卡簧钳是合拢握柄则钳口合拢，外卡簧钳是合拢握柄则钳口张开
3. 拉马
   - 作用：利用螺纹大压力把敲击配合的轴承拉出
   - 注意：分散压强
4. 拉铆枪
   - 作用：对铆钉抽芯
5. 角磨机
   - 作用：切割，打磨
6. 电钻
   - 作用：钻孔
7. 台虎钳
   - 作用：夹紧工件，利用夹紧力辅助装配拆卸
# 5.电机
1. 有刷电机
   - 原理：定子是磁铁（外），转子是通电线圈组（内），通电就能受磁场力旋转
   - 特点：价格便宜，但有碳刷摩擦，噪音大，易起火星，效率低，寿命短，需要维护保养
2. 无刷电机
   - 原理：转子是磁铁（外），定子是通电线圈组（内），通过霍尔传感器检测转子位置并给定子特定线圈通电吸引转子旋转
   - 特点：没有碳刷摩擦，噪音小，运行平稳，效率高，寿命长，但需要电子调速器
   - 参数：额定扭矩，额定转速，是否归零
   - 接线：
      1. 环控：闭环控制是一组三相线和一组排线，开环控制是一组三相线
      2. 电调：一组电源线，一组红和黑线的CAN信号线（少数是黄棕(S+-)和白黑(-S-)色的PWM信号线）
   - 设计：电调安装位置，走线孔位
   - 尺寸：
      1. 大疆2006
         - 类型：减速电机（无刷电机+减速箱）
         - 特点：效率高，精度高，噪音低。但扭矩低
         - 参数：额定扭矩1N·m，1:36减速比，电调C610
         - 设计：常用于小型拨弹弹盘、机械臂末端
      2. 大疆2508
         - 类型：减速电机（无刷电机+减速箱）
         - 特点：效率高，精度高，噪音低，扭矩高
         - 参数：额定扭矩4N·m，1:19减速比（会弄坏打印件），电调C620
         - 设计：常用于大型拨弹弹盘、摩擦轮
      3. 大疆6020
         - 类型：云台电机（无刷电机）
         - 特点：效率高，精度高，响应快，重量轻，内置电调
         - 参数：额定扭矩1N·m
         - 应用：舵轮底盘、云台转向
   - 特殊：
      1. 关节电机：
         - 特点：精度高，扭矩高，重复强，法兰输出
         - 尺寸：小米CyberGear，达妙DM-J8009，达妙DM-J4310-2EC
      2. 轮毂电机：
         - 特点：转子直接与车轮相连
         - 尺寸：领空9025
      3. 无人机动力电机：
         - 作用：驱动无人机螺旋桨产生足够升力克服重力
         - 特点：效率高，转速高
         - 尺寸：大疆E2000系统6010（PWM线），好盈X6plus（PWM线）
3. 伺服电机
   - 小型：舵机
      1. 参数：距离轴1cm位置能承载的重量kg/cm
      2. 特点：有转动角度范围。但精度差，扭矩小
   - 大型：工业伺服电机
