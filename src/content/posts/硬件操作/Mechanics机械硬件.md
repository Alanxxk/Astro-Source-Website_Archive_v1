# 2.标准件
## 2.1.紧固件
1. 螺栓螺钉：
   - 通名：配合螺母是螺栓，直接拧入是螺钉，日常统称是螺丝
   - 螺头形状分类：（螺头从厚到薄）
      1. 杯头：不易滑丝（推荐1）
      2. 薄头：螺头直径小和光滑
      3. 圆/扁/球头：圆形凸起，不会固连其他零件（推荐2）
      4. 法兰面：增大受力面积，防松
      5. 凸肩：短光轴长螺纹，柱形沉头定位
      6. 盘头：
      7. 倒边平头：微厚偏扁
      8. 平头：接触平面大。但容易滑丝（推荐4）
      9. 沉头：接触面积大，锥形定位。但要搭配沉头孔（推荐3）
      10. 紧定：没有螺头，尖尾，一般固定轴。但需要轴有凹槽，负载弱
      11. 顶丝：没有螺头，平尾，一般固定轴
   - 螺头花型分类：
      1. 十字：
      2. 一字：
      3. 内六角：小型机械，受力好，不易滑丝
      4. 外六角：大型机械
      5. 梅花
      6. 防盗：
   - 螺纹分类：（加工方式）
      1. 手工丝锥螺纹
      2. 车制螺纹
      3. 滚压螺纹
      4. 自攻螺钉自攻丝（无需螺母）
   - 螺牙分类：（空缺形状）
      1. 三角形：60°等边三角形（公制螺纹:单位mm,螺距），普通连接作用，代号M
      2. 管螺纹：55°等腰三角形（英制螺纹:单位英寸,螺纹牙数），管件连接作用，代号G
      3. 梯形螺纹：30°梯形，传递双向力矩，传动作用，代号Tr
      4. 锯齿形螺纹：30+3°锯齿，传递单向力矩，传动作用，代号B
      5. 矩形螺纹：矩形，非标准螺纹，传动作用
   - 其他参数分类：
      1. 公称直径：螺纹大径（大径-牙顶,小径-牙底,中径-中间值）
      2. 线数：单线，双线，多线
      3. 螺距：粗牙，细牙
      4. 导程：导程（旋转一圈位移）=线数×螺距（两相邻螺牙间距）
      5. 旋向：顺进逆出是右螺旋纹，无人机一般是左旋螺纹
      6. 牙型：全牙（全小径）、半牙（大径+小径）（位移小）
   - 选购参数：
      1. 公称直径：商品件常用M2-2.5，项目常用M3-5
      2. 长度：嵌入长度。贯穿通孔是零件高度+螺母高度+2mm，不贯穿盲孔是<孔深
      3. 性能等级：推荐镀镍，<8.8的普通螺栓，>8.8的低碳合金钢的高强度螺栓（小数点前数字×100=最小抗拉强度N/mm²,小数点后数字×小数点前数字×10=屈服强度N/mm²）。等级高则抗拉强、硬度高。但韧性差
      4. 制作精度：A（精制>8.8性能等级）、B（精制>8.8性能等级）、C（粗制-临时固定<8.8性能等级）
   - 设计：
      1. 避免两端螺纹
      2. 螺头螺母不与其他结构干涉
      3. 预留装配空间
      4. 开孔比螺丝大1-2mm（不能定位）
   - 装配：
      1. 螺栓+防松螺母(+普通螺纹)：螺栓长度要略超出螺母（强紧固）
      2. 螺栓+螺纹胶+普通螺纹：推荐中等强度可拆卸。靠近螺栓尾部的半螺纹涂抹螺纹胶（强紧固）
      3. 螺栓+弹簧垫圈+普通螺纹：（一般紧固）
2. 螺柱：
   - 作用：垫高，加强板件间距
   - 分类：（分为单通(公母)和双通(双母)）
      1. 尼龙：分类有圆柱形，六角柱。强度一般，不导电，不导磁
      2. 铝柱：分类有圆柱形，六边形头，六角柱。会形变
      3. 铜柱：强度一般，价格便宜，但精度差
      4. 不锈钢柱：强度高，但价格昂贵，精度差
3. 螺母：
   - 分类：
      1. 六角螺母：无防松效果
      2. 防松螺母：工程塑料附着在螺纹上，在螺栓和防松螺母配合时工程塑料会挤压产生压力
      3. 拉铆螺母：（螺母在外，在压铆机无法压铆和铆钉枪无法抽芯时使用）使用拉铆枪把螺母嵌入工件的孔
      4. K型螺母：爪子可以嵌入工件表面，一般在打印件使用，轻微损害零件
      5. 法兰螺母：凹凸法兰面帮助抓紧工件表面
      6. 船形螺母：（T型滑块螺母）一般卡在铝型材槽口
      7. 铜嵌螺母：（热熔螺母）加热3D打印件的孔等待膨胀变形后嵌入
      8. 六面螺母：六面都可以作为螺母，精度1mm，螺纹容易滑丝
      9. 蝶形螺母：手拧螺母，便于后续手调
      10. 夹板螺母：螺母和螺栓夹住工件
4. 垫圈垫片：
   - 分类：
      1. 弹簧垫片：回弹力防松（主要用于没有防松螺母位置），吸收冲击和振动
      2. 金属平垫：增大受力面积，补偿空间距离，分隔两个相对转动表面减少摩擦
      3. 尼龙垫片/柱：补偿空间距离，容易变形，不导电
      4. 止退垫片：
      5. 双叠防松垫片：
      6. 鞍形垫圈：回弹力防松
      7. 橡胶垫圈：密封，减震
      8. 绝缘垫圈：绝缘
5. 铆钉：
   - 材料：铝，钢
   - 特点：紧固强，重量轻，装配方便。但无法拆卸
   - 装配：通过铆钉枪抽芯，铆钉另一边会挤压变形而夹紧
   - 分类：
      1. 拉铆（拉铆枪）：铝制，可以用钻头暴力拆卸
      2. 敲铆（锤子敲）：不锈钢制，不可拆卸
6. 分体夹紧式法兰：
   - 结构：两部分金属弯片构成

## 2.2.传动件
1. 齿轮齿条
   - 定义：轮缘有齿，能连续啮合传递运动和动力
   - 参数：模数(齿大小)，齿数，齿厚
   - 设计：分度圆相切，渐开线，啮合线与切线，压力角
   - 公式：分度圆直径=模数×齿数，齿顶圆直径=模数×(齿数+2)
   - 装配：
      1. 直齿轮-直齿条：精度高，速度快，可靠强，承载力大，运动平稳，直线传动
      2. 直齿轮-直齿轮：效率高，传动比大（1-8），加工方便，价格便宜。但噪音大，重合度低
      3. 螺旋齿轮-螺旋齿轮：噪音小，运动平稳，重合度高，可高速传动。但会产生轴向力
      4. 圆锥直齿轮-圆锥直齿轮：交错轴传动。但要求安装精度高，轴承要承担轴向径向载荷，轴系设计复杂
      5. 圆锥螺旋齿轮-圆锥螺旋齿轮：交错轴传动，重合度高。但要求安装精度高，要求安装位置不易变形
2. 蜗杆蜗轮
3. 丝杆
   - 特点：精度高，运动平稳
   - 分类：
      1. 丝杆-梯形螺母：
         - 原理：相互螺旋运动实现直线运动
         - 特点：精度高，负载大，有自锁功能固定位置（条件是螺纹升角<摩擦角）。但效率低（摩擦阻力大）
      2. 丝杆-滚珠螺母：
         - 原理：滚珠和丝杆螺纹配合实现直线运动
         - 特点：效率高，精度高，负载大，噪音低，运动平稳，可高速传动。但价格昂贵，不能自锁，需要维护保养
4. 同步带-同步轮
   - 原理：主动轮通过同步带带动从动轮旋转
   - 特点：传动比准确，可传递转矩，可使用压板实现直线运动。但可能会跳齿，对中心距和尺寸稳定性要求高，需要使用张紧轮张紧同步带
5. 链条-链轮
   - 特点：负载大，距离长，可靠强。但效率低，需要使用张紧轮或楔形块张紧链条
6. 联轴器：
   - 作用：连接电机和传动机构，补偿两轴线不对中的误差，分担电机承受的载荷和冲击
   - 工作分类：
      1. 刚性联轴器：
         - 结构：两个楔形圆环和螺钉
         - 特点：负载大，精度高，拆卸方便，结构简单，价格便宜，两轴转速严格相等。但两轴严格对中，没有补偿相对位移能力，没有缓冲减震能力
         - 特殊：胀紧套（胀紧联结套）
            1. 原理：拧紧螺丝则孔内缩而轴外涨（注意额定扭矩），在内环与轴之间和在外环与轮毂之间都产生巨大抱紧力可以传递扭矩，实现轴和轮毂的无键连接
            2. 特点：负载大，可靠强，运动平稳，装拆方便，对中精度高
         - 尺寸：常用胀紧套，法兰联轴器（一端法兰一端孔,推荐不要用顶丝固定）
      2. 挠性联轴器：
         - 结构：两圆环和工程塑料（两圆环分别和工程塑料连接）/铝合金/弹簧/机械结构
         - 特点：两轴转速基本一致，可以补偿相对位移能力，一定缓冲减震能力。但负载小（负载太大会损坏弹性元件）
         - 尺寸：常用膜片联轴器（连接处有膜片,推荐用夹紧式），梅花弹性联轴器（连接处有塑料块,推荐用夹紧式），铝合金弹性联轴器（连接处是弹簧状,推荐用夹紧式），十字轴式万向联轴器（连接处是两个正交铰链,推荐用夹紧式）（补偿能力强但结构复杂和两轴转速可能不一致）
   - 装配分类：
      1. 胀紧联轴器：安装内环和轴不用拧紧，安装外环和轮毂后拧紧则楔形产生内外圈挤压力
      2. 夹紧联轴器：（缩小孔直径夹紧）通过高强度拉力螺栓夹紧轴实现连接，而外环和轮毂则固定，同理安装完成后再拧紧
      3. 顶丝联轴器：（扭紧螺丝顶紧轴,轴易滑和螺丝滑牙而不推荐）通过紧定螺钉压紧轴实现连接
7. 键
   - 作用：传递扭矩，旋转同步
   - 分类：
      1. 普通平键
      2. 半圆键
      3. 钩头楔键
      4. 外花键
      5. 内花键
8. 固定环
   - 分类：
      1. 光轴止动环：小轴，轴和环同步旋转止动
      2. 开口型固定环：大轴

## 2.3.支承件
1. 滚动轴承：
   - 作用：垫在旋转轴和旋转支持部分，把轴间原本的滑动摩擦力转化为轴承内球的滚动摩擦，滚动摩擦减少损耗而旋转顺畅
   - 结构：双侧橡胶密封圈2RS（防水但阻尼大）/金属防尘盖ZZ（无则润滑好），外圈，密封件，滚动体（球轴承,滚子）（单列,双列），保持架，内圈
   - 方向
      1. 轴向：沿轴的圆柱体旋转中心轴的方向
      2. 径向：沿轴的截面经过轴心的方向
   - 分类：
      1. 向心轴承：
         - 受力：主要径向受力（压力）
         - 结构：两个圆环和中间滚珠，不可分开
         - 尺寸：常见是深沟球轴承（例如带挡边的法兰轴承、薄壁轴承、微型轴承）（双列深沟球轴承,外球面球轴承）、调心滚子轴承、调心球轴承、圆柱滚子轴承
      2. 推力轴承：
         - 受力：只能轴向受力（推力）
         - 结构：两个圆环片（内外圈，硬质光滑工件可以省略但不推荐）和中间滚珠，外圈较大固定工件内圈较小配合轴，可分开
         - 尺寸：常见是推力球轴承、双向推力球轴承、推力滚针轴承、推力调心滚子轴承、推力圆柱滚子轴承
      3. 向心推力轴承：径向和轴向受力（压力+推力），常见是角接触轴承（双列角接触球轴承,四点接触球轴承）、圆锥滚子轴承、交叉滚子轴承
      4. 直线轴承：搭配线性导向轴使用，适用于直线运动导向机构
      5. 关节/鱼眼轴承：外球面和内球面，适用于速度低的摆动运动
   - 设计：轴工件不能同时碰到内外圈，尽量使用凸缘挡边法兰轴承（方便固定轴承位置），单向轴承推荐成对使用
   - 标准代号：轴承类型-尺寸型号-内径型号
   - 检查：相对滑动阻力很小，无晃动（过盈配合）（有晃动用680圆柱体胶固定），生锈用除锈剂，不够滑用润滑油
2. 滑动轴套：
   - 作用：厚度很薄保证只接触轴承内/外圈，内外圈光滑而减轻轴的磨损，减震减噪
   - 分类：
      1. 钢轴套：
      2. 卷制轴套：
      3. 无油石墨轴套：自润滑，可以作为滑动轴承
   - 装配：与孔过盈配合，与轴过渡配合

## 2.4.定位件
1. 卡簧挡圈：
   - 作用：对轴进行轴向限位（精度0.1-0.2mm），可以搭配波形垫圈补偿，可以承受一定力但大力会崩离崩断挡圈，对轴进行内外圈挡边设计，定位，止动
   - 分类：
      1. 轴用挡圈：C型挡圈和E型挡圈（无需卡簧钳），搭配轴使用，需要在轴上切割出卡簧槽
      2. 孔用挡圈：搭配孔使用，需要在孔上切割出卡簧槽
   - 装配：使用轴用卡簧钳、孔用卡簧钳安装
2. 销：
   - 特点：精度高，强度高
   - 分类：
      1. 圆柱销：定位，短轴传动
      2. 螺纹销：有内、外螺纹销，固定后光轴端用于导向定位
      3. 弹簧销：定位。不易变形，但难拆卸
      4. 开口销-销轴：轴向限位
3. 塞打螺钉：
   - 作用：转轴，定位销
   - 结构：螺头，长光轴（直径>大径），短螺纹
   - 特点：光轴精度高
   - 装配：搭配垫圈和防松螺母使用
4. 法兰
   - 作用：管道对接
5. 卡扣
   - 作用：抖动不掉，人手可掰开

## 2.5.导向件
1. 光轴：
   - 范围：轴套在光轴内
   - 特点：精度高，价格便宜，阻力均匀。但重量重，不易安装，需要两组光轴和导向座和直线轴承
2. 滑块滑轨：
   - 范围：滑块在滑轨内
   - 特点：精度高，阻力均匀。但重量重，需定制，价格昂贵，需要压住滑块使用，只能受较小侧向力
   - 装配：不能把滑块从滑轨上拿下来会掉钢珠，注意保养生锈
3. 抽屉滑轨：
   - 范围：可以伸出
   - 特点：负载大，价格便宜。但重量重，有虚位，阻力不均匀

## 2.6.弹缓件
1. 弹簧：
   - 分类：
      1. 压缩弹簧：抗压
      2. 拉伸弹簧：抗拉
      3. 扭转弹簧：抗扭
      4. 平面涡卷弹簧：恒力
2. 避震器
   - 作用：缓冲避震
3. 阻尼器
   - 作用：防震防抖动
   - 分类：
      1. 空气
      2. 弹簧
      3. 液压

# 3.非标件
## 3.1.结构件
1. 3D打印件
   - 格式：STL
   - 参数：
      1. 密度：过小则脆弱，过大则浪费
   - 分类：
      1. 加热头熔化材料再凝固
         - PLA（聚乳酸）：抗冲击好，抗层压好，力学性能一般，可嵌螺母（使用电烙铁）。但抗拉差，抗侧压差，受热易熔
         - PET：力学性能好。但表面粗糙
         - TPU：柔软可变形，抗冲击强
         - Nylon（尼龙）：力学性能较好。但表面粗糙
      2. 紫外光固化光敏树脂
         - 光固化：精度高，温度不敏感，表面光滑。但力学性能弱，柔软，长时间日光或荧光会软化变形，只能让加工商嵌入螺母
   - 注意：
      1. 壁厚不薄：否则影响力学性能，打印成败
      2. 孔留余量：由于打印机精度和冷却缩小的问题，孔要预设大一些
      3. 打印方向：层侧力学性能<层面
      4. 加强强度：可以塞入销等工件加强打印件强度
      5. 不印螺纹：因为材料硬度不够，所以不能打印螺纹孔用于固定，但可以打印放置螺母的孔位
2. 机加工件
   - 材料：常用6061铝合金
   - 加工：通过车钻镗铣刨磨和钣金和线切割等方式对材料进行去除多余的加工而得到工件
   - 特点：力学性能好，精度高，表面光滑，可加工螺纹孔。但加工图纸要标公差，价格昂贵
   - 设计：
      1. 孔离边远：避免刀头无法伸入加工
      2. 孔无斜面：孔的两侧都应该是平行面，避免钻入钻出受力不一致导致钻头偏斜
      3. 配合凸台：配合面要精加工处理但面积越大越贵，添加小凸台则只需要加工小凸台
      4. 孔深不长：孔深太长容易断钻头
3. 亚克力
   - 特点：透光，可以加工平面或平面弧。但强度低，材料软
   - 尺寸：常用厚度4、5mm
4. FR-4玻纤板
   - 颜色：黑，黄，绿
   - 特点：韧性好，耐燃好，重量轻，不导电。但抗弯差
   - 加工：雕铣机高速旋转铣刀平面加工
   - 尺寸：常用厚度2、3、4（结构件）、5mm
   - 设计：位置在受力小、抗冲击、接受微弯变形、拉板
5. 碳纤维板
   - 颜色：黑
   - 特点：力学性能好，强度高，重量轻，断面光滑。但价格昂贵，导电，外部阻燃内部可燃，加工粉尘有害，不能承受侧面打孔
   - 加工：打孔不要打柱形沉头孔而是锥形沉头孔（层间受力较弱）
6. 碳管
   - 尺寸：常用碳圆管，不常用碳方管（不易连接）
   - 特点：强度高，重量轻。但价格昂贵
   - 装配：通过夹紧方式连接
7. 铝方管
   - 尺寸：常用面积30×30mm厚度1mm
   - 特点：力学性能好，结构简单，价格便宜，重量轻。但抗压和耐热易变形，不可加工螺纹孔
   - 装配：

      2. 使用夹具夹紧铝方管后焊接
      3. 使用拉铆螺母铆接在铝方管上后有螺纹就可以用螺栓连接（注意拉铆螺母厚度）
   - 加强：塞入打印件，垫入玻纤板
8. 型铝

   - 特点：力学性能好，价格便宜。但重量重

# 4.机电件
## 4.1.动力件
1. 有刷电机
   - 原理：定子是磁铁（外），转子是通电线圈组（内），通电就能受磁场力旋转
   - 特点：价格便宜，但有碳刷摩擦，噪音大，易起火星，效率低，寿命短，需要维护保养

         - 设计：常用于大型拨弹弹盘、摩擦轮

# 6.设计
## 6.1.轴系设计
1. 欧拉角
   - Pitch：俯仰角，绕X轴旋转
   - Yaw：偏航角，绕Y轴旋转
   - Roll：翻滚角，绕Z轴旋转
## 6.2.结构设计
设计：固定后只保留需要自由度而其他不能动（轴向、径向、周向），使用轴承分隔开相互摩擦位置，保证结构强度，分析受力选择零件，清楚哪部分固定哪部分活动，安装位置余量，解耦模块化，考虑未来迭代的方便设计，强度（抵抗破坏）刚度（形变程度）精度（公差标注），避免应力集中（零件表面出现圆孔槽口,可以使用不锈钢垫片或玻纤板增加局部厚度），考虑螺丝疲劳（受压紧力和压力,薄板螺纹孔径3-5mm才够用），避免悬臂梁结构而是双端固定，考虑螺栓安装在孔里有活动余量所以要用定位销（定位销定位而螺栓是紧固）或者使用塞打螺栓
- 需求：结构件（强受力），连接件（弱受力），保护壳（受弯曲）
- 镂空：必要结构设计，减重轻量化，避免干涉其他零件安装，方便走线，通过镂空便于装配或维修或观察内部结构，要注意保留基础结构和主要受力支撑，推荐倒圆角的内三角形镂空
- 加固：使用其他结构件连接
## 6.3.紧固设计
1. 普通螺栓连接：螺栓+螺母
2. 铰制孔用螺栓连接：铰制孔螺栓+螺母
3. 双头螺柱连接：双头螺柱+螺母
4. 螺钉连接：螺丝+螺纹孔
5. 紧定螺钉连接：紧定螺钉+螺纹孔
## 6.4.加工设计
- 格式：DXF（正视零件导出）
- 加工：雕铣机切割平面板
   1. 内直角：圆柱刀头没有办法加工出内直角，可以在直角处割一个小圆槽（不推荐）
   2. 锐角：加工锐角时，回转时会产生力，可能会使板件翘起来，建议使用圆角过渡
   3. 孔位留余量：加工有误差，可以开大一点孔s
   4. 圆直径：圆直径不能<圆柱刀头

## 6.8.工件能力
1. 强度：抵抗破坏能力
2. 刚度：受力形变范围限制
3. 稳定：抵抗外界干扰能力
## 6.9.有限元分析
1. 应力：一点处的内力聚集程度，越集中该点越容易破损
2. 分析图：（小）蓝-绿-黄-红（大）
3. 应力集中：几何形状急剧变化位置，例如孔、槽、缺口、焊点
4. 解决办法：避免尖角，流线型外形，增加倒/圆角或渐变过渡
## 6.10.工件功能
1. 结构件：受力大，强度大，刚度大，4-6mm板件
2. 连接件：连接工件，刚度一般，2-3mm板件
3. 保护壳：1mm不镂空碳板，2mm镂空玻纤板
## 6.11.工件设计
1. 板件间拼插精度低（内角位置需要释放孔）
2. 孔位留余量（商加工1mm,自加工2mm）
3. 避免内角角度过小（在加工锐角时急停急转容易断刀）
4. 避免<2mm位置（刀具直径2mm）
