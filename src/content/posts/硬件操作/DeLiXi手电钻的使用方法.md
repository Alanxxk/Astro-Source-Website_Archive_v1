# 1.手电钻的说明
1. 手电钻品牌：德力西电气
2. 手电钻型号：DLX-KC01A
3. 供电来源：12V-2Ah的锂电池（可重复充电）
4. 照明功能：LED照明灯
5. 电机类型：有刷直流电机（带碳刷）
6. 调速方式：无极变速
7. 额定扭矩：22N·m（普通成年人使用螺丝刀扭矩≈4N·m）
8. 空载转速：1档是0-400r/min，2档是1500r/min
9.  空载时间：70min
10. 夹头类型：三爪式旋转夹头
11. 夹头直径：0.8-10mm
12. 旋转功能：起拧螺丝，钻孔模式
13. 适用范围：拧螺丝，木材钻孔，塑料钻孔

# 2.手电钻的使用
1. 更换电池：在手电钻的弹匣处，按住电池缺口往下拉，换成新电池往上推装入
2. 更换钻头：手握住枪口旋转头，逆时针旋转旋开夹头，将批头放入夹头，顺时针旋转夹住工具
3. 正反旋转：在扳机位置后上方处，推动塑料片，左突则顺时针转（紧螺丝/钻孔），右突则逆时针转（松螺丝/出孔），双突是卡住不能旋转（锁定功能）
4. 调节速度：根据按压扳机按钮深度程度，调节钻头旋转速度，越深越快
5. 钻头模式：在枪上方，拨动拨钮，显示1挡是低转速高扭矩（适合硬材料钻孔），显示2挡是高转速低扭矩（适合软材料的精细操作）
6. 调节扭力：在枪口旋转头后，旋转模式可以调整扭力（可以握住枪口旋转头感受每一档的扭力,手电钻弹跳则扭力低），1-5是拧螺丝，7-9是钻木材，11-13是钻瓷砖，15-17是钻砖头，19-25是钻钢材，横线是装饰线（无作用），子弹螺纹图案是平钻钻孔（在其他手电钻型号上,小锤子图案是冲击钻功能,螺丝图案是松紧螺丝功能）

# 3.手电钻的批头
1. 钻柄：半圆，三角，圆两槽
2. 钻头：三角钻/偏心钻，霸王钻，麻花钻（含钴），攻丝钻，三尖钻，沉孔钻，沉头钻，平翼钻，孔锯，三尖扁钻，支罗钻，拉花钻，圆珠刀，宝塔钻
3. 尺寸：
   - 标准：6，8，10
   - 非标：5，7，9
4. 使用：
   - 钻金属：低转速慢避免发热，大扭矩带来足够切削力，可以先钻小孔再钻大孔
   - 钻木头/塑料/亚力克：通常是带尖钻头

# 4.后续更换电钻事项
1. 品牌推荐
   -  第三梯队：威克士，史丹利，德力西，小米，锐奇，绿林
   -  第二梯队：东成，大有，大艺
   -  第一梯队：博世，世达，美沃奇，得伟，牧田，麦太保，高壹工机
2. 参数选择
   - 噪声
   - 同心度
   - 有无线/电池（+续航时间）
   - 有无刷电机
   - 扭矩-电压
   - 扩展功能
   - 安全离合（阻塞停机）
3. 功能分类（能力低到高排序）
   - 电动螺丝刀：匀速旋转，作用是拧螺丝
   - 手电钻：匀速旋转，作用是拧螺丝和开孔
   - 冲击起子：边转边间断施加瞬间旋转力，作用是拧长螺丝
   - 冲击钻：边转边前后微小高频震动冲击，作用是在硬材料中开孔
   - 电锤：前后往复运动锤击/边转边前后往复运动锤击，作用是在混凝土级别的超硬材料中开孔
