# 2.硬件控制
## 2.1.钥匙
1. 按键
   - 图标：在关闭驾驶室车门后按下一次是所有车门上锁，在上锁后长按是关闭所有车窗和遮阳板和玻璃天窗，驻车空调
   - 解锁：按下一次是解锁驾驶室车门，按下两次是解锁全部车门，在解锁后长按是打开所有车窗和遮阳板和玻璃天窗
   - 后备箱：在后备箱关闭时长按约1s打开后备箱上门，在后备箱打开时长按关闭后备箱上门
   - 菱形：在后备箱关闭时长按约1s打开后备箱下门，在后备箱打开时长按关闭后备箱下门，回家照明灯、开关后备箱、后备箱入口
2. 电池
   - 型号：纽扣电池
   - 更换：按住背部扣键，前推和侧面取开装饰板，拉出电池，更换电池，插回装回
3. 机械钥匙
   - 取出：按住背部扣键，前推和侧面取开装饰板，弯开机械钥匙
   - 开门：一手拉住车门把手，一手向外按压盖罩，取下盖罩，插入机械钥匙上锁或解锁
4. 识别（钥匙在身边）
   - 故障：在没有识别到车钥匙时，将钥匙背面放在方向盘右后方缺口处
   - 解锁：完全握住前车门把手，车钥匙靠近已上锁车辆1.5m
   - 上锁：在关闭车门后用手指触摸前车门把手内凹槽约1s，车钥匙远离已解锁车辆3m
   - 关窗：在关闭车门后用手指触摸前车门把手内凹槽
   - 声音：在行驶状态上锁会响两声喇叭
5. 其他钥匙
   - KeyCard：略（P74-78）
   - 数字钥匙：略（P74-79）
## 2.2.后备箱
1. 开关：
   - 钥匙按键：按下钥匙的后备箱按键约1s打开后备箱上门，在后备箱上门打开后按下钥匙的菱形按键约1s打开后备箱下门。长按则关门
   - 后备箱门按键：在解锁车辆后，按压后备箱上门按键打开后备箱上门，在后备箱上门打开后可以按下后备箱下门按键打开后备箱下门。在确保下门没有装载物后按下后备箱下门按键关闭后备箱下门，按下后备箱上门车辆按键关闭后备箱上下门，按下后备箱上门闭锁按键关闭后备箱上下门并给车辆上锁（要驾驶室车门关闭和钥匙在后备箱区域）

## 2.5.侧车门
1. 把手位
   - 前面：记忆功能（座椅位置、外后视镜、方向盘高度、前挡玻璃抬头显示高度）
      1. 保存设置：调整位置，按下SET按键LED亮起，按下要保存的数字按键发出保存成功音效
      2. 调出设置：按下对应数字按键
   - 后面：所有车门锁
2. 推门位
   - 前面：在钥匙在车内时，按下是打开后备箱上门，上拉是关闭后备箱上下门
   - 前中：调整外后视镜（在故障时可以手动按压镜面玻璃来调整外后视镜）
   - 中前左：展收外后视镜（在>40km/h外后视镜自动展开）
   - 中前右：在拨钮推到驾驶室位置和倒车档R时，开启外后视镜向下倾斜便于观察路沿，拨钮推到副驾驶位置则关闭
   - 中后：所有车窗升降
   - 后面：车窗的安全开关

## 2.6.方向盘
1. 圆盘外
   - 左后上：换低档拨片
   - 右后上：换高档拨片
   - 左后下：按下按键移动方向盘调整高度
   - 左上：
      1. 上拨：右转向信号灯
      2. 下拨：左转向信号灯
      3. 前拨：开启远光灯（固定）
      4. 后拨：开启远光灯（松开则无）
      5. 顶上键：控制行驶数据
      6. 顶下键：自动切换远近光灯
   - 正上：屏幕仪表，前挡玻璃抬头显示
   - 右上：雨刮器
      1. 上拨：前挡雨刮静止位置，上拨是自动感应雨量开启雨刮，再上拨是雨刮器普通速度，再上拨是雨刮器快速速度
      2. 下拨：点动刮水
      3. 后拨：喷水刮水
      4. 中滑块：往上是提高自动感应雨量的灵敏度，往下是降低（传感器在车内后视镜前方）
      5. 外旋钮：后窗雨刮静止位置，后旋是喷水刮水，前旋是间歇雨刮，再前旋是喷水刮水
2. 圆盘内
   - 左按键：
      1. 左上RESUME：使用设置值继续定速巡航控制系统
      2. 右上CANCEL：中断定速巡航控制系统
      3. 正中拨钮：调整巡航车速
      4. 左下SET：保存速度（当前速度/手动设速）
      5. 正下I/O：启停定速巡航控制系统
      6. 右下MODE：选择定速巡航控制系统（手动车速/巡航控制/车距控制/辅助驾驶模式）
   - 正中间：
      1. 车标：按下是喇叭
   - 右按键：
      1. 左上-：减少音量
      2. 右上+：增加音量
      3. 正左◀：前切电台曲目
      4. 正中滚轮：控制屏幕仪表和前挡玻璃抬头显示的配置，左右倾斜/滚轮/按下则切换/选择/选中选项
      5. 正右▶：后切电台曲目
      6. 左下电话：按下是接听手机电话/使用手机拨打号码
      7. 正下列表：排列屏幕仪表内容布局
      8. 右下麦克风：启停语音输入（要设置）

## 2.8.座椅
1. 侧边按键
   - 前上：按下前键是靠背宽度减小，按下后键是靠背宽度增大
   - 前下：按下前键是腰撑加大，按下后键是腰撑减小，按下上键是腰撑上移，按下下键是腰撑下移
   - 中间：前后
   - 后上：肩部靠背
   - 后下：靠背倾斜度
2. 校准座椅
   - 问题：电动座椅不准确会在显示屏显示信息
   - 设置：长按侧键前移座位直至停止并再次长按侧键前移直至再次停止
3. 头枕
   - 向上：按下侧键，上推头枕
   - 向下：按下侧键，下推头枕
   - 向前：按下枕键，前推头枕
   - 向后：按下枕键，后推头枕
4. 头枕拆卸
   - 注意：头枕都不能拆卸
## 2.9.屏下键
1. 拨钮
   - 作用：调整风道
2. 按键
   - 旋钮：旋转旋钮则调节音量，按下旋钮则开关声音
   - |◀◀：按下一次是切换电台曲目，长按是曲目快退
   - ▶▶|：按下一次是切换电台曲目，长按是曲目快进
   - ▲：故障警示闪烁灯
   - MAX：除霜
   - REAR：加热后窗玻璃
## 2.10.档位区
1. 按键
   - 左正上：动态控制系统
   - 左中上：泊车辅助系统
   - 正左：在踩下刹车时按下是启动/熄停汽车
   - 左中下：驾驶模式（SPORT运动、COMFORT舒适、ECO-PRO节能）
   - 左下中：下坡车速控制系统
   - 左正下：自动驻车功能Auto-H
   - 正上：换档拨片（R倒车、N空档、D普通前进、S运动模式），档位显示
   - 正中：驻车档P按键（在熄车时会自动挂入，在行驶时会自动松开）
   - 正下：（要设置）在熄车时自动拉起手刹，（无需设置）在起步时自动放下手刹
   - 右上：
      1. HOME：调出主菜单界面
      2. MEDIA：调出媒体收音机、切换媒体源界面
      3. MAP：调出导航地图界面
      4. TEL：调出电话界面
      5. NAV：调出输入导航目的地界面
   - 正右：控制器
      1. 旋转：切换选项
      2. 按下：选择选项
      3. 左推：返回上一个界面
      4. 右推：切换界面、切换选项
      5. 触控板：（要设置）
   - 右下：
      1. BACK：返回上一个界面
      2. OPTION：调出选项界面
## 2.11.车顶棚
1. 按键

   - 后：紧急呼叫SOS
2. 拨钮
   - 中：前后拨是关开玻璃天窗

## 2.12.特殊控制
1. 通道式洗车
   - 方法：挂入档位区的N空档，踩下刹车踏板，按下档位区的熄车按键，车辆可以滑行但在35min后自动挂入P驻车档
2. 故障拖离
   - 方法：快速按下按下档位区的启动/熄停按键三次，踩下刹车踏板，挂入档位区的N空档
3. 手动档
   - 开启：在档位区的D普通驾驶时拉动方向盘后方的换档拨片则进入临时手动模式（在无换档操作且匀速行驶一段时间后自动切回自动档），在档位区的S运动模式时拉动方向盘后方的换档拨片则进入持续手动模式
   - 解除：拉住左侧和右侧换档拨片则切换回自动档
4. 起跑控制模式
   - 作用：获得起跑最佳加速
   - 问题：给零部件高负荷而磨损，在起跑过程中不能转动方向盘，仅适合在干燥不打滑环境下和在发动机行驶了10km暖机后使用
   - 方法：启动汽车，按下档位区的左正上按键，中控屏启动TRACTION，挂入档位区的S运动模式，左脚用力踩刹车踏板直至踩尽并保持，右脚调节油门踏板得到合适的起步转速并等待发动机转速恒定，观察到屏幕仪表有旗帜图标并在亮起3s内松开刹车踏板，车辆加速，尽快再重新按下起跑控制模式按键
5. 驾驶模式
   - COMFORT：在运动型驾驶模式和节省电能模式中均衡
   - SPORT：动态调整设置，优化底盘实现高灵敏度
   - ECO-PRO：节省电能
6. 自动驻车
   - 方法：在车辆行驶时，按下档位区的自动驻车Auto-H按键，刹停车辆则自动拉起手刹，踩下油门踏板则自动放下手刹
7. 更换雨刮
   - 翻开：在启动车辆后，方向盘圆盘外右上杆往下拨直至雨刮器停住，从前挡风玻璃上完全翻开雨刮器
   - 折叠：更换完成后，折叠雨刮器到前挡风玻璃上，下拨直至雨刮器返回静止位置
8. 动态稳定控制系统
   - 作用：提高车辆稳定性
   - 设置：按下档位区的左正上按键，稳定DSC-OFF，按下档位区的左正上按键重启
9. 动态牵引力控制系统
   - 作用：在特殊路况车辆稳定性受限时，保证足够的牵引力
   - 设置：按下档位区的左正上按键，牵引力TRACTION，按下档位区的左正上按键重启
10. 下坡车速控制
   - 设置：按下档位区左下中按键，调整车速是方向盘左侧正中拨钮或踩下刹车踏板


# 3.软件控制
## 3.1.中控屏按钮
1. 圆圈箭头
   - 作用：导航界面
   - 功能：输入目的地、查看路况信息、设置地图点
2. 风扇
   - 作用：温度界面
   - 功能：设置所有空调
3. 苹果CarPlay
   - 作用：导航界面
   - 功能：与苹果手机连接使用Apple-CarPlay
4. 房子
   - 作用：主菜单界面
   - 功能：返回主菜单
5. 四格
   - 作用：导航系统
   - 功能：访问应用、设置车辆功能
6. 电话
   - 作用：通讯界面
   - 功能：访问电话讯息、连接管理移动设备
7. 音符
   - 作用：娱乐界面
   - 功能：收音机、外部设备
## 3.2.中控屏不常用设置
1. 显示屏
   - 开启；车辆解锁、点按屏幕
   - 关闭：车辆长时间没有输入操作，用手指从上往下滑动-选择关闭屏幕
2. 调节亮度
   - 设置：应用菜单-车辆-显示器-控制显示器-夜间亮度
3. 语音回应
   - 设置：应用菜单-车辆-系统设置-音效
4. 提示窗口
   - 设置：应用菜单-车辆-系统设置-提示窗口
5. 快速访问
   - 开启：用手指从上往下滑动
   - 关闭：用手指从下往上滑动
   - 添加：长按所需功能-选择添加为快捷方式
   - 删除：打开快速访问-长按所需功能-选择删除快捷方式
6. 触控板
   - 作用：开关档位区的控制器的触控板
   - 设置：应用菜单-车辆-系统设置-触控板
   - 使用：手写字符、左滑是删除字符、滑动地图、轻点是显示菜单
7. 调整主菜单
   - 设置：在主菜单界面下，在屏幕右边缘左滑，可以设置主菜单显示
   - 排序：按住应用图标移动
   - 地图：手指滑动是移动地图，双指缩放是缩放地图，轻点是显示菜单
8. 语言语音
   - 系统语言：应用菜单-车辆-系统设置-语言
   - 语音语言：应用菜单-车辆-系统设置-语音控制-个人助理-对话语言
   - 语音声音：应用菜单-车辆-系统设置-语音控制-个人助理-声音
   - 语音回复：应用菜单-车辆-系统设置-语音控制-个人助理-回答长度
   - 语音打断：应用菜单-车辆-系统设置-语音控制-个人助理-允许用户打断系统说话
   - 在线语音：应用菜单-车辆-系统设置-语音控制-个人助理-使用在线语音处理
   - 语音可视：应用菜单-车辆-系统设置-语音控制-个人助理-形象
   - 三方语音：应用菜单-车辆-系统设置-语音控制-长按
   - 语音音量：在语音提示过程中旋转屏下键的旋钮
   - 手机语音：（要通过Apple-CarPlay连接苹果手机）
   - 自动惯例：应用菜单-全部应用-智能偏好记忆
9. 手势控制
   - 开关：应用菜单-车辆-系统设置-手势控制-手势控制
   - 设置：应用菜单-车辆-系统设置-手势控制
   - 位置：车内后视镜下方=显示屏位置
   - 示例：
      1. 食指前后：接听电话、确认弹出窗口
      2. 整手划走：拒接电话、关闭弹出窗口、结束语音
      3. 食指顺圈：增大音量
      4. 食指逆圈：降低音量
      5. 捏拇食指整手左右移动：在停车时，转动车辆视图、转动摄像头视图
      6. 拇指向左拳头：返回上一个界面、播放上一首歌
      7. 拇指向右拳头：前进一个界面、播放下一首歌
      8. 未分配的个性化手势：食指中指指向显示器、张手-握拳-张手
   - 个性：应用菜单-车辆-系统设置-手势控制-选择个性化手势
   - 快捷：长按菜单项，添加为手势快捷方式
10. 软件升级
   - 下载分类：自动下载，手动下载
   - 手动下载：应用菜单-全部应用-系统设置-远程软件升级-搜索升级
   - 当前版本：应用菜单-全部应用-系统设置-远程软件升级
