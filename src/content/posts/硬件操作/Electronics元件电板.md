# 2.控板编程
## 2.1.原理
1. 作用：控制电路实现控制目标元件，尽量快稳准
2. 芯片：
   - CPU：解释计算机指令和处理计算机数据（计算机一部分）
   - GPU：绘图运算（计算机一部分）
   - NPU：专用于人工神经网络硬件加速（计算机一部分）
   - MPU：解释计算机指令和处理计算机数据的微型芯片（计算机一部分）
      1. 系统：Linux，Android
   - MCU：集成电路单芯片微型控制器（基本是完整计算机）
      1. 特点：体积小，耗电低，电路简单，价格便宜。但运行频率低
      2. 寄存器：二进制寄存器存储数据
         - 4位：简单家电控制
         - 8位/16位：复杂家电控制
         - 32位：使用嵌入式操作系统控制
      3. 系统：无，FreeRTOS
      4. 编译：高级编程语言不能直接在单片机中执行，需要编译转换成二进制的机器语言。C语言效率高代码精简，因而成为最适合语言
      5. 烧录：二进制的机器语言可执行文件传输到单片机芯片上
      6. 历史：
         - 单片机只能烧录一次，使用大电流把晶体管烧断
         - 使用紫外线和激光擦除烧录数据
         - 晶体管更换为电容，可以擦除和烧录多次
3. 系统：
   - 分类：
      1. 裸机系统：直接写应用代码
         - 特点：代码耦合性强，拓展性差
      2. GPOS：通用操作系统
         - 例子：Windows、Linux
         - 特点：注重运算能力。但不保证实时响应
      3. RTOS：实时操作系统
         - 例子：FreeRTOS（开源免费）、RT-Thread
         - 特点：稳定性高，内存占用小，保证在规定时间完成任务。但不注重运算能力
         - 状态：
            1. 就绪：任务等待运行代码指令
            2. 运行：任务正在执行代码指令
            3. 阻塞：暂停任务
            4. 挂起
            5. 僵尸：任务所有代码指令已执行完毕
4. 端口：
   - VCC：整个电路工作的供电高电压
   - VDD：设备的工作供电高电压
   - VEE：设备工作的供电低电压/负电压
   - VSS：设备工作的供电公共低电压，一般是0V
   - CLK：Clock时钟信号，各种设备都按照这个固定周期脉冲信号同步频率、处理数据、数据传输
   - RST：Reset复位，重新开始程序
   - GND：Ground地线，0V
5. 英文：
   - Pin：引脚
   - Periphera：外部设备
6. 控环：（基本是完整计算机）
   - 开环控制：无反馈值，易受干扰
   - 闭环控制：有反馈值，能抗干扰，输出精度高，但需要编码传感器检测
      1. PID控制：
         - 问题：从当前位置快速到达固定的目标位置
         - 控制：角度，转速，扭矩(≈驱动力≈加速度)，温度，电压，电流……
         - 原理：输入→(反馈测量值)→测量值和输入值处理结果(误差)→比例p、积分i、微分d调节→执行机构→输出→(测量)
         - 数值：
            1. 目标值：目标位置
            2. 反馈值：测量位置
            3. 输出量：U=控制力
            4. 误差：E=目标位置-测量位置
            5. 稳态误差：在近似稳定后，目标位置和测量位置仍存在的差值
         - 情况：一个目标值一个反馈值
         - 公式：U(k)=Kp·E(k)+Ki·ΣE(i)+Kd·\[E(k)-E(k-1)]，k是离散时间且时间间隔相同（在时间间隔不相同时可以使用近似面积≈积分）
            1. E：
               - 当前误差：E(k)=目标值-本次测量值
               - 上一次误差：E(k-1)=目标值-上一次测量值
            2. K：超参数，自行设置数值
            3. Kp·E(k)：对于位置的线性控制
               - 作用：快速减小误差
               - 特点：Kp系数过大导致在目标位置附近振荡(超调)但可以减小基本稳定后位置误差(稳态误差)，Kp系数过小导致在靠近目标位置时到达目标位置速度太慢=稳态误差太大和抗阻力能力太差和
            4. Ki·ΣE(i)：对于位置的细微控制(稳态误差控制)
               - 作用：快速减小稳态误差(根据次数累加误差值)
               - 特点：Ki系数过大导致在目标位置附近振荡(超调)，Kp·E(k)的数值过小导致到达目标位置速度太慢
               - 动态优化：Ki=Ki÷(1+E(i))，在误差大时减小Ki
            5. Kd·\[E(k)-E(k-1)]：对于速度的阻尼控制
               - 作用：抑制超调(控制误差曲线的加速度斜率从而抑制速度缓慢到达目标位置实现抑制超调)，但减慢到达目标位置的速度
               - 特点：在Kd系数过大而阻力过大时，可以抑制在目标位置附近振荡(超调)但到达目标位置的速度太慢；在Kd系数过小而阻力过小时，无法抑制在目标位置附近振荡也会导致到达目标位置的速度太慢
      2. 微分先行PID控制
         - 问题：从当前位置快速到达频繁徘徊的目标位置，目标位置跳跃点的误差变化很大(阶跃)，此时PID的微分d调节(误差斜率)会受较大影响
         - 情况：一个目标值一个反馈值
         - 原理：微分d调节只受测量值影响而不受输入值影响=微分d调节:误差斜率→测量值斜率
         - 公式：U(k)=Kp·E(k)+Ki·ΣE(i)+Kd·\[测量(k-1)-测量(k)]，（测量值前-后是保持原微分d调节的误差值相减的负号）
      3. 双环串级PID
         - 问题：从当前位置快速到达固定的目标位置
         - 情况：两个目标值两个反馈值，两个控制量有微积分的数学关系
         - 原理：外PID嵌套内PID
7. 引脚：
   - 名称：GPIO=General Purpose Input/Output=通用输入输出
   - 参数：
      1. 上下拉
         - NO Pull-Up-Down：不启用内部上拉下拉电阻，但引脚会悬空容易受到干扰，需要外部电路保持电平稳定。≈外部设备无电源
         - Pull-Up：启用内部上拉电阻，在引脚悬空时电阻拉到高电平
         - Pull-Down：启用下拉电阻，在引脚悬空时电阻拉到低电平
      2. 输入
         - ：上拉输入。在芯片引脚没有输入时，读取引脚是高电平
         - ：下拉输入。在芯片引脚没有输入时，读取引脚是低电平
         - ：浮空输入。在芯片引脚没有输入时，读取引脚是不确定电平
         - ：模拟输入。芯片引脚可以输入模拟/离散数字信号
      3. 输出
         - Push-Pull：推挽输出。强力直接驱动输出高/低电平（推挽输出电平遇到相反电平会烧坏）
         - Open-Drain：开漏输出。只能驱动低电平，而高电平需要依赖外部上拉电阻(电阻越小输出切高电平越快)，在不驱动低电平时引脚处于悬空状态。常用在I2C总线
         - ：复用推挽输出
         - ：复用开漏输出
      4. 电平
         - Low：默认输出低电平
         - High：默认输出高电平
      5. 出速
         - Low：引脚输出电平变化速度是低速，可以减少电磁干扰和功耗，一般是LED或继电器
         - Medium：中速输出，一般应用
         - High：高速输出，一般是驱动数据总线或PWM信号
         - Very High：超高速输出，一般是高速数字通信或高频信号
      6. 数模
         - ADC：外部模拟信号转换为数字信号
         - DAC：数字信号转换为外部模拟信号
8. 中断
   - 方式：
      1. 轮询：每隔一段时间访问端口查看信号，有则触发中断
      2. 打断：在接收到信号后触发中断
   - 过程：请求，响应，处理，结束(回到原来执行位置)
   - 特点：
      1. 暂停：突发时间暂停主程序执行中断代码，在执行完成后可以返回主程序暂停处继续执行
      2. 嵌套：在执行一个中断代码过程中可以执行另一个中断
   - 优先级：
      1. 作用：在多个中断同时执行时，根据中断的抢占优先级（=比较优先级）数值而插队和响应优先级（=子优先级）数值而排队
      2. 数值：数字越小而优先级越高
      3. 触发：只比自己数字小的优先级才会触发中断
9.  定时器
   - 原理：检测时钟源产生的等距方波信号的上升沿或下降沿进行计数
   - 作用：
      1. 计时/定时中断（总方波数×时间间隔）
      2. PWM（一个周期中高低电平比例,在高频时表现是平均值）（通过比较计数值实现高低电平切换）
      3. 传输数据的时间间隔
   - 分类：
      1. 通用定时器
      2. 基本定时器
      3. 高级定时器
   - 寄存器：
      1. PSC分频倍数：定时器接收到的信号频率太高，所以倍数分频(÷)后开始计数。寄存器实际值=填写数值=目标数值-1
      2. CNT计数器：对定时器倍数分频后的信号计数(一般是+1)，计数到最大值后清零
      3. ARR计数总数/自动重装器：存储计数目标数字，计数器达到该数字则计数器清零。寄存器实际值=填写数值=目标数值-1
      4. CRR捕获比较：在输入时捕获数值，在输出时存储比较计数值数字。数值在代码中设置
   - 时钟源：
      1. 芯片内部：芯片内部RC振荡电路（电容电感充放电电路,容易受到环境影响而不稳定）
         - HSI：高速内部时钟，频率不稳定
         - LSI：低速内部时钟，频率不稳定，看门狗
      2. 外部晶振：使用两个引脚连接芯片（石英晶体制成晶体振荡器）
         - HSE：高速外部时钟，频率稳定
         - LSE：低速外部时钟，频率稳定
   - 倍分频：时钟源产生的频率信号，可以自主选择是否经过PLLCLK的倍分频(×÷)得到SYSCLK信号，SYSCLK信号经过AHBP的倍数分频(÷)后得到HCLK信号给主要内部设备使用（主要的时钟总线=主频≈心跳），HCLK经过AHBP+数字的倍分频(×÷)可以得到PCLK信号给TIM定时器、通信时钟波特率、其他外设使用
   - 通道：一个定时器有多个输出通道，每个通道设定一个芯片引脚
   - 输出：（N是取反）
      1. 输出比较
      2. PWM生成：
         - 原理：脉冲宽度调制，使用离散变化数字信号+时间代替连续变化模拟信号
         - 控制：使用定时器计数+设定CCR捕获比较值，在计数<比较值则输出高电平，在计数>比较值则输出低电平。
         - 公式：占空比=高电平时间/周期时间=CCR捕获比较值/ARR计数总数目标数值，分辨率=占空比变化最小间隔
      3. 强制输出
10. 模数转换
   - 分类：
      1. ADC：模数转换器
      2. DAC：数模转换器
11. 通信：
   - 方式：
      1. 串行：传输使用一根线，每次传输一位
      2. 并行：传输使用多根线，每根线每次传输一位
      3. 同步：需要收发双方时钟频率一致，需要添加一根时钟线发送脉冲实现同步
      4. 异步：规定传输波特率而双方使用各自时钟，使用数据帧的起始位和停止位实现同步
      5. 单工：单向传输
      6. 半双工：双向传输，但双方不能同时发送/接收
      7. 全双工：双向传输，双方可以同时发送/接收
   - 速度：
      1. 比特率：传输二进制bit位数据/秒
      2. 波特率：传输信息bit位数据/秒
   - 分类：
      1. USART串口
         - 通信：串行异步全双工
         - 对象：一对一
         - 速度：按bit位收发数据
         - 端口：Tx发送，Rx接收，GND地线（提供参考电平0V而消除噪声）
         - 接线：Tx接Rx，Rx接Tx，GND接GND
         - 信号：
            1. 高电平：1
            2. 低电平：0
         - 波特率：推荐921600
         - 数据帧：
            1. 起始位：1位低电平（高变低）
            2. 数据位：0-7位高/低电平数据
            3. 校验位：根据数据位计算的结果发送1位高/低电平，接收方依此判断传输无误。常见有无校验，奇校验，偶校验
            4. 停止位：一般是1位高电平
         - 传输：
            1. CPU：内存←→CPU(中断)(占用资源)←→寄存器←→传输数据线←→外部设备
            2. DMA：内存←→DMA←→寄存器←→传输数据线←→外部设备
      2. CAN
         - 通信：串行异步半双工
         - 对象：多对多（一根总线连接多设备）
         - 接线：两根缠绕线（相互抵消外界干扰），红对红（CANH,正极），黑对黑（CANL,负极）
         - 方向：
            1. 同向：在缠绕线捋直后，两端的端子卡扣是同向
            2. 反向：在缠绕线捋直后，两端的端子卡扣是反向
         - 稳定：使用60Ω终端电阻(2个120Ω电阻并联)连接CANH和CANL
         - 传输：
            1. CAN节点：设备
            2. CAN总线：CANH，CANL
         - 信号：
            1. 类型：差分信号
            2. 隐性电平：在CAN节点的High线-Low线=0时（浮空产生），输出信号=1
            3. 显性电平：在CAN节点的High线-Low线>0时（有电压差），输出信号=0
            4. 总线电平：在总线同时存在显隐电平时，显性电平会覆盖隐性电平，可以用来确定通信设备优先级
         - 波特率：最高1Mbps，再多容易丢包
         - 数据帧：
            1. 数据：多个字节 
            2. 分类：
               - 数据帧：帧起始(1位)+仲裁场(12位)+控制场(6位)+数据场(0-8字节)+CRC场(16位)+应答场(2位)+帧结尾(7位)
                  1. 仲裁场：12位ID标识符。在总线上同时有多个信息时，ID越小→0越多→显性电平越多→优先级越高
                  2. 控制场：数据场的数据长度
                  3. CRC场：校验，总线控制
               - 遥控帧
               - 错误帧
               - 过载帧
               - 帧间隔
         - 软件过滤器：
            1. 作用：设备总是判断总线信号与自身有关会浪费时间，可以通过CAN设备手册的ID和反馈报文标识符(设备反馈给芯片)和控制报文标识符(芯片控制设备)辅助区分信号（电机可以调整拨码开关可以调整ID从而影响标识符）
            2. 模式：
               - 列表模式：制作ID表，数据ID在表中才接收
               - 掩码模式：通过设置掩码，掩码和数据每位独立相乘得到的结果就是屏蔽码，使用屏蔽码和验证码比较一致则成功
         - 硬件外设：
            1. 发送：邮箱
            2. 接收：队列(FIFO=First-In-First-Out)
      3. I2C/IIC
      4. SPI

## 2.2.控板类型
1. Arduino
   - 软件：Arduino
   - 硬件：Arduino板，专用USB串口线
2. 51单片机
   - 软件：Keil，STC-ISP烧录软件（代码写入硬件芯片），CH340-CH341驱动
   - 硬件：51单片机，通用USB串口线
   - 分类：
      1. STC89C51RC
      2. STC89C52RC
      3. STC89C53RC
      4. ……
   - 
3. ESP32
   - 厂家：Espressif乐鑫（芯片是32位的MCU,单核双核）
   - 特点：芯片集成无线通信模块
4. STM32
   - 厂家：STMicroelectronics意法半导体
   - 代号：
      1. 家族：
         - STM8：ARM核心8位的MCU，单核（同时间只能执行一条指令）
         - STM32：ARM核心32位的MCU/MPU，单核（同时间只能执行一条指令）
      2. 产品：
         - A：汽车级
         - F：基础型
         - L：超低功耗
         - S：标准型
         - WB：无线产品
         - H：高性能
         - G：主流型
      3. 功能：
         - STM8系列：
            1. 103：入门级
            2. F52：汽车级
            3. L31：低端汽车级
         - STM32系列：
            1. 051：入门级
            2. 103：基础型
            3. 303：升级版，更多外设
            4. 407：高性能
            5. 723：超强性能
            6. 152：超低功耗
      4. 引脚数量：
         - T：36脚
         - C：48脚
         - R：64脚
         - V：100脚
         - Z：144脚
      5. 闪存容量：SRAM(变量)，FLASH(常量+代码)
         - 4：16K字节的闪存存储器
         - 6：32K
         - 8：64K
         - B：128K
         - C：256K
         - D：384K
         - E：512K
      6. 芯片封装：
         - H：BGA球栅阵列(底面多点焊接)（适合引脚多芯片,但难手工焊接）
         - T：LQFP(四边突出引脚焊接)（方便焊接）
         - U：VFQFPN/QFN(底面四边引脚)（省空间,但较难焊接）
         - Y：WLCSP64/CSP（类似BGA也难手工焊接）

   - 作用：多个任务近似同时进行(不断快速轮着切换任务)
   - 优先级：
      1. 分类：
         - 空闲
         - 低
         - 低于正常
         - 正常
         - 高于正常
         - 高
         - 实时
      2. 特性：
         - 抢占：在高优先级任务触发时，直接抢占CPU而挂起暂停低优先级任务
         - 同级：在多个相同优先级任务时，按照任务就绪先后执行，每达到一个时间片（=频率周期）会自动强制轮询任务
   - 任务状态：
      1. 就绪：任务等待分配CPU
      2. 阻塞：任务不能使用CPU
   - 配置方法：
      1. 工具：在STM32CubeMX中，选择中间件和软件包的FreeRTOS
      2. 版本：FreeRTOS选择CMSIS_V1版本，打开任务与队列

   - 代码解析：
      1. 任务句柄：≈任务名称
      2. 函数声明：STM32CubeMX自动在FreeRTOS.c声明任务函数，而根据设置在FreeRTOS.c或其他文件手动定义函数
      3. 创建任务：STM32CubeMX会自动在MX_FREERTOS_Init中根据任务参数去创建任务得到任务句柄
      4. 定义任务：自己创建.c和.h文件声明和定义任务函数，在定义函数时同理使用externC变量+函数，使用while保持任务持续运行

# 3.电板设计
## 3.1.简介
1. 名称：PCB（Printed Circuit Board）印刷电路板
2. 软件：EDA（Electronic Design Automation）电子设计自动化软件
3. 作用：芯片引脚和外设连接起来实现芯片使用外设
4. 历史：
   - 发明：真空管，真空二极管，往真空二极管加入栅极得到三极真空管
   - 设备：利用插件元件布置和焊接得到机器
   - 专利：金属箔切割成线路导体，线路上下面贴上石蜡纸，线路交点设置导通孔连接不同层间
   - 树脂：改进了酚醛树脂的生产技术以得以工业化生产
   - PCB：在绝缘的基板印刷线图案，通过电镀方式成功建立导体，但当时电子管元件体积大发热量大而并未广泛应用
   - 化蚀：印刷电路开始被广泛接受，开启化学蚀刻工艺，使用金属箔腐蚀法得到PCB，现在常见的是铜板腐蚀
   - 人机：随着技术发展，PCB设计工作从纯人工变成计算机辅助(EDA软件)，PCB制造工作也从人工变成机械，过孔也从全通孔变成盲埋孔等等复杂的孔型
5. 流程：需求→方案→软硬设计→样品验证→批量生产

2. 半导器件
   - 二极管：
      1. 作用：交流电整流，信号检波，防反接，稳压
      2. 特性：二极管的伏安特性
         - 正向特性：在电压正接时，0-aV死区是改变电压而无电流（锗管是a=0.1V,硅管是a=0.5V），a-bV导通区是电压与电流基本是线性关系但存在导通压降损耗（锗管是0.2-0.3V,硅管是0.6-0.8V）
         - 反向特性：在电压反接时，0-cV截止区是改变电压而小电流不变，c-dV击穿区是电压超过击穿cV而电流变化幅度大（遏制反向击穿电压增大）
      3. 符号：三角形是正极，竖线是负极，电流方向是正极到负极（三角形指着的方向）
      4. 极性：短引脚是负极，三角形指向方向是负极，颜色点是负极，竖线是负极
      5. 参数：击穿电压，工作电流
      6. 分类：
         - 普通二极管：
            1. 特点：单向导电

         - 稳压二极管（齐纳二极管）：
            1. 特点：利用反向击穿特性使得改变电流而电压基本不变

         - 发光二极管：
            1. 作用：发光灯=LED灯
            2. 符号：双箭头朝外
            3. 极性：大结构是负极
            4. 应用：照明，显示
         - 光电二极管：
            1. 作用：
            2. 符号：双箭头朝内
         - 变容二极管：
            1. 特点：电容随着反向电压变化，电容量较小
            2. 符号：一条竖线变成两条竖线
            3. 应用：高频电路的自动调谐调频调相
         - 肖特基二极管：
            1. 特点：反向恢复时间很短，正向导通压降很小
            2. 符号：竖线有弯钩小尾巴
            3. 应用：高频低压大电流的整流续流保护检波
         - 瞬态电压抑制器（TVS）：
            1. 特点：耗散高瞬态功率浪涌
            2. 符号：两个相对的稳压二极管符号
            3. 应用：抑制高压尖峰

   - 三极管：
      1. 应用：放大电流电压，无触点开关（B基极电流控制）
      2. 特点：成本高，噪声大，热稳定性差，抗辐射能力弱
      3. 封装：
         - TO插件：极性需要查阅手册
         - SOT贴片：在俯视图中，中间是C集电极，左边是B基极，左边是E发射极
      4. 结构：两个PN结（单向导电性），中间是B基极(控制)，两侧是C集电极和E发射极
         - NPN：
            1. 特点：在B基极电压 > E发射极电压时，C集电极电流能到达E发射极
            2. 符号：从竖线指向外面（P指向N）≈B是P
         - PNP：
            1. 特点：在B基极电压 < E发射极电压时，E发射极电流能到达C集电极
            2. 符号：从外面指向竖线（P指向N）≈B是N

         - 状态：
            1. 静止：在三极管的B基极E发射极之间的电压U~BC~(U~B~>U~E~) < PN结导通电压时，Ibe=0，Ice电流很小
            2. 放大：在三极管的B基极E发射极之间的电压U~BC~(U~B~>U~E~) > PN结导通电压，三极管的B基极C集电极之间的电压U~BC~(U~B~<U~C~) < PN结导通电压时，Ic=K×Ib（K放大倍数需要查阅元件手册）。——C集电极和B基极的PN结没有导通但C集电极在吸收B基极和E发射极之间电流的电子，B基极和E发射极导通
            3. 饱和：在三极管的B基极E发射极之间的电压U~BE~(U~B~>U~E~) > PN结导通电压，三极管的B基极C集电极之间的电压U~BC~(U~B~>U~C~) > PN结导通电压时，U~CE~≈A×I~C~。——C集电极和B基极的PN结导通，B基极和E发射极的PN结导通，得到C集电极和E发射极近似导通

      1. 应用：无触点开关（G栅极电压控制）
      2. 原理：调节场效应管的控制电压（施加电压发生场效应），可以控制电路通过电流
      3. 特点：成本低，噪声小，热稳定性好，抗辐射能力强

         - 增强型：在U~GS~=0时，Id=0，需要加电压才导通
         - 耗尽型：在U~GS~=0时，Id≠0，需要加反向电压才关断
      6. 结构：
         - 结型场效应管（J-FET）
         - 金属氧化物半导体场效应管（MOS-FET）
      7. 封装：
         - TO:常见型号TO252（适用<100V,<70A），TO220(安装散热片)、TO263(背接散热焊盘)（适用<中压120A/高压20A）

1. 原理图：
   - 电容：
      1. 噪声信号经过接地电容后得到平稳信号

      2. 双滤波电容：大电容滤低频(电源先经过),小电容滤高频(电源后经过)

2. 电板布置：
   - 要求：有序排列，分布均匀，模块同区，芯片晶振屏幕复位放中央，人为使用元件放边缘，注意元件三维模型干涉
   - 布件：元件要对齐和等间距和翻转便于连线(线尽量不要交叉)，丝印要翻转正视，晶振要靠近芯片和便于等长布线，图层可以顶层换成底层，并联电容则小容值靠近芯片电源引脚，电源的滤波电容靠近芯片引脚
   - 布线：
      1. 优先顶层其次底层
      2. 信号线正常(10mil)而电源线加粗(>40mil-2.3A)
      3. 走线>90°推荐135°(锐角在腐蚀环节产生酸角效应导致线宽变窄和产生阻抗不连续导致影响信号传输和产生天线效应导致传输高频信号对外产生干扰)
      4. 高频信号线尽量短和其他信号隔离(禁止布线层+接地过孔屏蔽(法拉第笼原理))
      5. 避免相邻层近似重合平行走高频信号线(串扰)(至少平行距离>3倍线宽)(推荐正交走线)
      6. 远离安装孔和板边(避免加工切掉导线)
      7. 焊盘或过孔与导线连接部位/T型或十字导线连接部位添加泪滴(平滑过渡,连接稳固)
      8. 密度优先(元件密集区域)和关键元件优先(次要元件布线顾全整体)和关键信号线优先(质量要求高优先布线)
      9. 尽量横平竖直而不要长倾斜布线
      10. 长圆形焊盘尽量在短边出线(不会形成直角)
      11. 晶振高频信号线要求等长
      12. 尽量不要在一个元件的焊盘之间走线但可以换层再走
      13. 线宽<=焊盘出线边的宽度
      14. 差分双线尽量短和紧密走线而避免干扰和信号衰减，差分信号是关键信号优先绘制，差分信号尽量不超过两队过孔避免增加线路的寄生电感而影响信号完整性，双线长度相差<5mil保证信号同步，和其他信号网络/地线距离>20mil

      2. 铺铜：铜箔块
         - 作用：大电流、散热
         - 位置：电源、共地
         - 方式：自动与导线相隔一段距离并填满区域
         - 分类
            1. 填充：方形铜皮，作用是大电流
            2. 区块：多边形铜皮
      3. 过孔：
         - 作用：连接不同层电路，定位固定电路板
         - 分类：（价格从低到高）
            1. 通孔：上层到底层，机械钻孔
            2. 盲孔：上层/底层到内层，钻孔/电镀
            3. 埋孔：内层到内层，钻孔
      4. 焊盘：
         - 作用：裸露铜皮显示元件焊接位置
         - 形状：
            1. 通孔式焊盘：焊接固定插件元件
            2. 表贴式焊盘：焊接固定贴片元件
         - 功能：
            1. 隔热焊盘：方形四边中间有空隙，作用是避免焊接时铜皮散热降温
            2. 直连焊盘：圆形，作用在定位固定无需焊接/大电流焊盘
            3. 散热焊盘：
      5. 丝印：
         - 作用：显示文字图案帮助标识元件位置
      6. 阻焊
         - 作用：铜层覆盖油墨遮住不需要焊接的线路，露出焊盘（阻焊>焊盘）
      7. 叠层：
         - 组成：PP半固态片(半固态树脂材料+玻璃纤维)，Core芯板
         - 分类：
            1. 双层：顶层(信号+电源+地)，底层(信号+电源+地)。使用通孔连接
            2. 四层：顶层信号层，电源层，地层，底层信号层。使用通孔连接
            3. 六层：顶层信号层，地层，信号层，电源层，地层，底层信号层。使用过孔连接
         - 设计：
            1. 导电层：信号(导线,铺铜)，
            2. 掩膜层：丝印(标注)，阻焊(裸铜)，锡浆(开钢网的孔)
            3. 绘辅层：板框(板子尺寸)，多层(过孔)，机械(元件装配尺寸)，3D外壳(装PCB板外壳尺寸)
      8. 元件封装
         - 作用：原理图的元件符号和元件实物引脚的对应，辅助标识元件放置方向
   - 步骤：新建PCB→设置板框→元件布局→连接布线→检查DRC(DesignRuleCheck)→导出Gerber(生产文件)→下单制作

1. STC89C51
   - 封装：LQFP(正方形)，DIP(长方形)
   - 最小系统：
      1. 电路：数据手册
      2. 引脚：XTAL晶振，RST复位，VCC和GND电源
      3. 设计：芯片+晶振电路(振荡电路放大输出信号)+复位电路(重置初始化系统状态)+电源电路(芯片通电)
   - 工作方式：
      1. 准双向口：P1234
      2. 仅输入
      3. 开漏输出：P0，做IO口则外加上拉电阻，做地址数据总线则不需要外加
   - 元件：
      1. 芯片：LQFP-STC89C51（工作电流mA毫安级别）
      2. 供电口：Type-C
         - 6Pin：仅供电
         - 16Pin：USB2.0
         - 24Pin：USB3.0
      3. 开关：SS3235S
      4. 稳压器：LDO-AMS1117（5V→3.3V）
      5. 按键：微动贴片按键
      6. 指示灯：0805贴片LED
      7. 排针：HDR-M-2.54
      8. 晶振：ABLS-12MHZ
   - 原理图：
      1. Type-C：6Pin
         - VBUS：电源正极。连接VCC+5V
         - GND：电源负极。接地
         - CC：快充。先连接5.1K电阻再接地
         - EH：固定针脚。接地
      2. SS3235S：一侧全接GND（不使用），一侧凸边接VCC+5V而剩下两个接+5V
      3. AMS1117：
         - VOUT：相连，单边VOUT接+3.3V，单边接10uF和100nF并联电容(电容末端接地)
         - VIN：接+5V，接10uF和100nF并联电容(电容末端接地)
         - GND：接地
      4. LED：正极接+3.3V和1K电阻，负极接地
      5. ABLS：根据芯片数据手册使用元件连接晶振电路
      6. 芯片：VCC接+5V和接100nF电容(电容末端接地)，GND接地，XTAL连接晶振输出，根据芯片数据手册使用元件连接复位电路(电容并联开关)，剩余每个引脚添加网络标签并更改为单片机引脚名称
      7. P0口：每个P0引脚分别接一个上拉电阻和+5V
      8. HDR-M：接芯片引脚，接电源接地引脚，接串口+接电源接地引脚
      9. 开关：开关并联电容(消除按键抖动)，一端接芯片引脚，一端接地
      10. LED：负极接芯片(芯片输入>输出能力)，正极接1K电阻和+3.3V
      11. 模块分类：上侧栏-放置-矩形
      12. 模块注释：上侧栏-放置-文本
      13. 制图信息：
      14. DRC检查：下侧栏-DRC-检查DRC
      15. 导出：上侧栏-导出-物料清单BOM
   - 电路板：
      1. 转图：原理图转PCB
      2. 板框：右侧栏-图层-板框层，放置-线条-矩形，板框一个端点在原点上，框选调整宽高参数，框选-右键-添加-添加圆角
      3. 定孔：四端点位置添加通孔
      4. 固定：框选-右键-锁定
      5. 模块：右侧栏-图层-顶层，在原理图框选模块使用布局传递分离开元件模块
      6. 隐线：工程设计-网络-飞线-隐藏GND
      7. 布件：在PCB板框上布局元件
      8. 信电布线：布线连接元件。布线顺序是信号线>电源线>地线铺铜。同理隐藏电源正极线，从芯片引脚开始布线，10mil信号线，晶振周围选择24-12milGND的过孔20mil连线屏蔽+放置-禁止区域-多边形-围住芯片晶振引脚和晶振信号线和<地孔线晶振区域-多层图层和禁止铺铜。显示电源线进行布线，根据焊盘调整电源线宽

      10. 丝印：右侧栏-图层-顶层丝印层(可以隐藏底面和顶层)，上侧栏-放置-文本，调整文本的线宽和高度，文本对齐和等间距，排针的丝印要框起来，开关标记开启方向(小实心圆)，元件功能，芯片型号，引脚号码
      11. Logo：右侧栏-图层-顶层丝印层，上侧栏-放置-图片
      12. DRC：下侧栏-DRC-检查DRC，上侧栏-设计-实时DRC
      13. 导出：上侧栏-导出-PCB制板文件Gerber
2. USB拓展坞
   - 名称：Universal Serial Bus通用串行总线
   - 版速：
      1. USB1.0：1.5Mbps
      2. USB1.1：12Mbps
      3. USB2.0：480Mbps
      4. USB3.0/3.1Gen1/3.2Gen1：5Gbps
      5. USB3.1/3.1Gen2/3.2Gen2×1：10Gbps
      6. USB3.2/3.2Gen2×2：20Gbps
      7. USB4.0：40Gbps
   - 接口：
      1. 方形：USB Type-A
      2. 方梯形：USB Type-B
      3. 长圆形：USB Type-C
      4. 扁梯形：MicroUSB
   - 传输：差分信号
      1. 结果：两条信号线的电平差
      2. 优点：放大有效信号，屏蔽噪声信号(抑制共模噪声)
      3. 线型：短则并行，长则双绞线（保证噪声同时耦合到两根线上）
      4. 相对：单端信号是有信号线和参考基准线，但抗干扰能力差
   - 元件：
      1. 芯片：SL2.1A(USB2.0)
         - 上行口：DP(信号线正极)，DM(信号线负极)
         - 下行口：DP+数字，DM+数字
         - 接晶振：XOUT，XIN
         - 接电源：VDD5，GND
      2. USB接口：
         - 180度短体直边Type-A母座：VCC，D+(信号线正极)，D-(信号线)，GND，SHELL(固定插针)
         - 16PinType-C：VBUS(接两个焊盘)，DP(信号线正极)，DM(信号线负极)，GND(接两个焊盘)，CC(快充,不使用)，SBU(备用引脚,不使用)，EP(固定插针)……
      3. 晶振：Crystal-12MHZ
      4. 保险丝：1.5A快恢复保险丝1206
      5. 指示灯：LED
   - 原理图：
      1. SL2.1A：参考数据手册的示例电路，VDD接100nF接地后接+5V，GND接地，CC接5.1K电阻后接地，剩余每个引脚添加网络标签并更改为单片机引脚名称
      2. Crystal：接芯片的XIN和XOUT
      3. Type-C：DP接芯片的DP，DP接DP，DN接芯片的DM，DN接DN，SBU放×，VBUS接10uF和100nF并联电容连地给VCC+5V供电，GND和EP接地
      4. Type-A：D+接芯片的DP，D-接芯片的DM，VCC接10uF和100nF并联电容连地给+5V供电，GND和SHELL接地
      5. 1206：接VCC+5V和+5V
      6. LED：接+5V和2K电阻后接地
      7. 细节：同理模块分类，模块注释，制图信息，DRC检查，导出
   - 电路板：
      1. 制板：同理转图，板框，定孔，固定，模块，隐线
      2. 布件：建议USB接口略微超出板框(方便后续安装外壳)
      3. 信电线：调整参数:设计-设计规则-规则管理-差分对-差分对长度误差=5。添加差分对:工程设计-差分对-+-选择网络-选择焊盘-芯片到5个USB和1个晶振。差分对布线:布线-差分对布线。等长调节布线：布线-等长调节-选择两线。22mil连线+5V焊盘。同理晶振周围GND过孔和粗线连接，内部放置禁止布线多边形的多层禁止铺铜
      4. 铺铜过孔：VCC电源铺铜能增加通过大电流能力。填充区域-多边形-两个电源焊盘-顶层。放置三个(增强过流能力)24-12mil的VCC过孔走底层连接VCC电源铺铜。同理顶层放置靠近GND焊盘的GND过孔阵列，框选板框的顶层和底层进行GND铺铜
      5. 丝印：右侧栏-过滤-隐藏铺铜区域，添加顶层丝印层的文本作为丝印。同理添加Logo和功能介绍(可以反向填充文本背景)
      6. DRC：同理检查DRC
